# 实验八 · 综合数据处理模型构建（ModelBuilder + 水文分析）

> 对应理论：空间分析（建模 / 水文）。参考教材第十一章水文分析。用 **ModelBuilder** 把整条水文分析链自动化。

## 一、实验目的
掌握**集水区划分**的基本流程，掌握 **ModelBuilder** 的使用。

## 二、实验内容
用 ArcGIS ModelBuilder 搭建集水区划分模型。集水区划分（参考教材第十一章水文分析）可分为：**填挖、汇流量计算、水流长度、河网提取、流域分割**等步骤。

## 三、关键操作（步骤）
1. **新建模型**：ArcToolbox 中先 `New Toolbox`（**模型只能存放在 Toolbox 中，不能存放在文件夹**，否则报错 `Models can only be saved in a toolbox`）；再右键工具箱 `New ▸ Model`，保存命名 `Watershed`。
2. 将工具依次拖入模型窗口并连线，设置输入输出：
   ```
   DEM.tif
     └▶ Fill（填洼）                     → DEM_Fill
        └▶ Flow Direction（流向）         → FlowDir
           └▶ Flow Accumulation（汇流累积）→ FlowAcc
              └▶ Raster Calculator         → Con("%FlowAcc%" > 500, 1)   # 阈值提取河网
                 ├▶ Stream Link            → StreamL
                 │   └▶ Watershed          → (流向 FlowDir + 倾泻点 StreamL, Pour point field=VALUE)
                 │       └▶ Raster to Polygon → Basin（矢量集水区）
                 └▶ Stream to Feature      → Stream（矢量河网）
   ```
3. 运行整个模型，得到河网与流域分割结果。

## 四、注意事项（必考细节）
- **Raster Calculator 阈值**：`Con("%FlowAcc%" > 500, 1)`——汇流累积大于 500 的栅格判定为河道。
- **Stream to Feature**：把栅格河网转为**矢量河网**。
- **Watershed 参数**：Input flow direction = `FlowDir`；Pour point data = `StreamL`；Pour point field = `VALUE`。
- **Raster to Polygon**：把栅格集水区转为**矢量**（Basin）。
- **ModelBuilder 价值**：流程可视化、可重用、可批处理（变量参数化），是 GIS 自动化建模核心。

## 五、思考题
在生态环境保护中如何使用「集水区」这一概念？
> 参考：集水区（流域）是水文与物质迁移的天然单元——可用于面源污染负荷核算、水源地保护区划定、水土流失与生态修复分区、流域生态补偿与统一管理等。
