# 实验四 · 专题地图和地图制图输出

> 对应理论：第 4 章 空间数据的可视化表达。把分析结果「画成图、出成版」。

## 一、实验目的
掌握 ArcMap 绘制**专题地图**的基本流程与**制图输出**方法。

## 二、实验内容
绘制「枫树坝水库水质采样地图」：主图为水库与采样点（S1–S8），并附**索引图 / 区位图（Locator / Inset Map，中国—珠江流域）**，完成整饰与导出。
> 指导书说明：具体绘制步骤参考理论课程「空间数据的可视化表达」相应章节。

## 三、关键操作（步骤）
1. **数据符号化（Symbology）**，按数据性质选四类专题表达：
   - 单一符号（Single symbol）
   - 唯一值 / 分类（Categories / Unique values）
   - 分级色彩 / 分级符号（Graduated colors / symbols，定量分级）
   - 比率 / 点密度 / 统计图表（Proportional / Dot density / Charts）
2. 切换**布局视图（Layout View）**，设置页面与方向。
3. 插入整饰要素：`Insert ▸` 图名（Title）、**图例（Legend）**、**比例尺（Scale Bar）**、**指北针（North Arrow）**、格网（Grid）。
4. 制作**索引图**：新增一个数据框（Data Frame），放入小比例尺底图，框出主图范围。
5. **导出**：`File ▸ Export Map`，选 JPG / PNG / PDF / EPS / TIFF，设 DPI（出版常用 300 dpi）。

## 四、考点与要义
- **专题地图四类符号化方法**及适用：定性用唯一值，定量用分级 / 比率。
- **地图整饰五要素**：图名、图例、比例尺、指北针、（数学要素/格网）；外加责任要素（编制者、坐标系）。
- **数据框（Data Frame）**：一个布局可含多个数据框，用于主图 + 索引图。
- **导出格式**：栅格（JPG/PNG/TIFF）vs 矢量（PDF/EPS/AI），出版选矢量或高 DPI 栅格。
