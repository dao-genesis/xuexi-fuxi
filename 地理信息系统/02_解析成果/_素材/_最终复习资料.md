> 素材：[_第01章_地理信息系统_Geographic_Information_System,_GI.md](./_第01章_地理信息系统_Geographic_Information_System,_GI.md) · 主版 90 页

#### 一、核心概念（名词解释）

- [x] **地理信息系统（GIS）**：既是跨地球科学、空间科学、信息科学和计算机科学的一门应用基础学科，又是一项工程应用技术。以地学原理为依托，在计算机软硬件支持下，研究空间数据的采集、处理、存储、管理、分析、建模和显示。
- [x] **地理数据**：与地理环境要素有关的事物/现象的数量、质量、分布特征、联系和规律等的数字、文字、图像和图形总称。
- [x] **地理信息**：有关地理实体和地理现象的性质、特征和运动/变化状态的表征和一切有用的知识，是对地理数据的解释。
- [x] **数据与信息的关系**：信息来源于数据，是数据内涵的意义；信息是数据的表达，数据是信息的载体。
- [x] **地理数据的特征**：空间上的分布性、时间上的序列性、数量上的海量性、载体的多样性、位置和属性的对应性。
- [x] **地理信息的特征**：空间相关性、空间层次性、空间区域性、空间多样性。
- [x] **GIS的构成**：五大部分——计算机硬件系统、软件系统、空间数据、地学模型、应用人员。
- [x] **GIS基本特征**：数据的空间定位特征、空间关系处理的复杂性、海量数据管理的能力；GIS是空间数据和属性数据的综合体。
- [x] **空间分析**：GIS的核心功能，也是GIS与其他计算机系统的根本区别。

#### 二、关键公式 / 模型

- [x] **RUSLE公式**（通用土壤流失方程）：A = R·K·L·S·C·P，用于土壤侵蚀量估算
- [x] **C-Fix模型**：植被固碳量估算模型
- [x] **GIS功能链**：数据采集与输入 → 数据编辑与更新 → 数据存储与管理 → 空间查询与分析 → 数据与图形的交互显示 → 应用模型与系统开发

#### 三、重要案例 / 实验 / 例题

- [x] **John Snow霍乱溯源**（1854伦敦）：空间数据可视化+空间插值，确认Broad Street水井为霍乱源头
- [x] **COVID-19疫情地图**：数据空间可视化
- [x] **苏州河重金属空间分布**：采样点地图+空间插值+污染物浓度可视化
- [x] **东江上游土壤侵蚀**：RUSLE模型+时序变化分析
- [x] **农村排水模式优化**：基于GIS的计算框架——空间聚类→空间叠置→管网优化→处理设施优选→造价核算

#### 四、高频考点（速记）

1. GIS的定义（必考）：跨学科应用基础+工程技术，研究空间数据采集→显示全链路
2. GIS五大部分构成：硬件、软件、数据、模型、人员
3. GIS核心功能 = 空间分析（区别于其他系统的根本标志）
4. GIS是空间数据+属性数据的综合体
5. 地理数据三大特征：空间分布性、时间序列性、海量性
6. 数据库建设占系统投资70%
7. GIS软件三类：支撑软件（OS/DB）→平台软件（ArcGIS/QGIS）→应用软件（二次开发）

#### 五、思考题 / 自测

- [x] 题：GIS与一般信息系统的本质区别是什么？
  答：GIS处理空间数据和属性数据的综合体，核心在于空间分析功能。
- [x] 题：为什么说空间分析是GIS的核心功能？
  答：空间分析是GIS区别于其他计算机系统的根本区别，也是评价GIS功能的主要特征。
- [x] 题：GIS在环境科学中有哪些应用？
  答：污染物空间可视化、土壤侵蚀评估、水质监测、排水模式优化等。

### 第 2 章 · 空间数据-知识点回顾

> 素材：[_第02章_空间数据-知识点回顾.md](./_第02章_空间数据-知识点回顾.md) · 主版 169 页

> 基于 OCR 全文自动提取，人工校核待补。

#### 一、核心概念（名词解释）

- [x] **空间数据采集**：将地图、外业观测、航空相片、遥感图像、文本资料等转为计算机可识别的数字形式
- [x] **地理空间数据**：表示空间实体的位置、形状、大小及分布特征的数据
- [x] **非地理空间数据（属性数据）**：表示空间实体的属性特征，是对地理空间数据的说明
- [x] **矢量数据**：用点、线、面及其拓扑关系表达空间实体；空间拓扑关系清晰，易于网络分析；图形数据质量高
- [x] **栅格数据**：用规则网格表示空间实体；数据量大，易执行空间分析，与遥感数据匹配
- [x] **矢量与栅格对比**：矢量——拓扑清晰、图形精美、叠置难；栅格——易分析、数据量大、图形粗糙
- [x] **空间数据库**：数据统一定义、组织和存储，集中管理，消除"数据孤岛"；采用复杂数据模型（层次/网络/关系/对象）
- [x] **Geodatabase**：ArcGIS第三代数据模型，包含要素类（点/线/面）、要素数据集、关系表
- [x] **Shapefile**：由主文件(.shp)、索引文件(.shx)、dBASE表(.dbf)、空间索引组成
- [x] **拓扑关系**：对现实世界地理事物之间空间关系的模仿程度；用于检查数据错误

#### 二、关键公式 / 模型

- [x] **栅格编码方法**：直接栅格编码（最简单最直接）、游程编码、四叉树编码
- [x] **矢量数据结构**：点——坐标(X,Y)；线——坐标序列；面——闭合坐标序列+拓扑关系
- [x] **数据模型层次**：层次模型→网络模型→关系模型→对象模型

#### 三、重要案例 / 实验 / 例题

- [x] **MXD文件操作**：数据层排序原则（点线面由上至下、重要程度由上至下、线划由下至上、色彩由下至上）
- [x] **创建要素类**：Catalog→选中要素数据集→新建→选择类型（Point/Line/Polygon）
- [x] **矢量数据编辑**：选择要素→裁剪/分割/整形/移动
- [x] **属性编辑**：添加字段→设置类型（Text/Short/Long/Double等）→计算字段值

#### 四、高频考点（速记）

1. 矢量 vs 栅格数据优缺点对比（必考）
2. 空间数据采集方法：野外采集、地图数字化、摄影测量、遥感处理
3. Geodatabase三级结构：要素类→要素数据集→Geodatabase
4. Shapefile组成：.shp+.shx+.dbf（至少3文件）
5. 拓扑关系用于检查数据错误
6. 数据层排序四原则
7. Geoprocessing环境设置影响工具运行

#### 五、思考题 / 自测

- [x] 题：矢量数据和栅格数据各有何优缺点？
  答：矢量——拓扑清晰、图形精美、叠置难、投影变换易；栅格——易分析、数据量大、图形粗糙、与遥感匹配好
- [x] 题：空间数据库与文件系统有何本质区别？
  答：数据统一定义组织存储、集中管理消除数据孤岛、采用复杂数据模型
- [x] 题：如何在ArcGIS中创建要素类？
  答：Catalog→选中Geodatabase/要素数据集→新建→选择类型→定义字段和坐标系

### 第 3 章 · 空间数据的转换与处理

> 素材：[_第03章_空间数据的转换与处理.md](./_第03章_空间数据的转换与处理.md) · 主版 104 页

> 基于 OCR 全文自动提取，人工校核待补。

#### 一、核心概念（名词解释）

- [x] **空间参考系统**：确定地理目标平面位置和高程的平面坐标系和高程系的统称
- [x] **地球椭球体**：地球表面几何模型四类——自然表面→大地水准面→地球椭球面→数学模型
- [x] **参考椭球体**：与局部地区大地水准面最佳拟合的地球椭球体，需确定定位与定向
- [x] **大地坐标系**：以参考椭球面为基准面建立的坐标系，用大地经度L(-180~180)、大地纬度B(-90~+90)、大地高程表示
- [x] **北京54坐标系**：参心坐标系，克拉索夫斯基椭球体，大地原点在前苏联普尔科沃；缺点——椭球参数不精确、与我国大地水准面西高东低倾斜
- [x] **西安80坐标系**：参心坐标系，四个椭球基本参数，多点定位，大地原点在西安；定向明确
- [x] **CGCS2000**：地心坐标系，2008年7月起启用，北斗卫星导航使用此系
- [x] **WGS84**：地心坐标系，GPS定位结果所属坐标系
- [x] **高程基准**：我国主要——1956黄海高程系、1985国家高程基准（青岛大港验潮站）
- [x] **地图投影**：将地球椭球面上的经纬网按一定数学法则转换到平面上；实质是一一对应的映射关系
- [x] **投影变形**：长度变形、面积变形、角度变形；投影必然产生变形
- [x] **高斯-克吕格投影**：横轴切圆柱投影，中央经线无变形，我国大中比例尺地图常用
- [x] **兰勃特投影**：正轴割圆锥投影，适用于中纬度东西延伸区域
- [x] **空间变换**：同一区域不同空间参考系统之间的转换，包括坐标变换和投影变换
- [x] **数据格式转换**：不同GIS平台间数据格式互转（Shapefile↔Geodatabase↔Coverage等）

#### 二、关键公式 / 模型

- [x] **投影变形类型**：长度比 m = 实际长度/椭球面长度；面积比 p = 实际面积/椭球面面积
- [x] **高斯投影分带**：6°带（1:2.5万及更小比例尺）和3°带（1:1万及更大比例尺）
- [x] **坐标转换方法**：四参数（平面相似变换）、七参数（三维赫尔默特变换）

#### 三、重要案例 / 实验 / 例题

- [x] **ArcGIS定义投影**：ArcToolbox→Data Management→Projections and Transformations→Define Projection
- [x] **ArcGIS投影变换**：Project工具，需选择转换方法（如Beijing54→CGCS2000的转换参数）
- [x] **中国大地原点**：陕西省泾阳县永乐镇北流村（北纬34°32'27.00″东经108°55'25.00″）

#### 四、高频考点（速记）

1. 四大坐标系统对比（北京54/西安80/CGCS2000/WGS84）——参心vs地心是核心区别
2. 地球表面四层模型：自然表面→大地水准面→椭球面→数学模型
3. 地图投影必然产生变形（长度/面积/角度）
4. 高斯-克吕格投影：横轴切圆柱、中央经线无变形、6°/3°分带
5. 我国高程基准：1985国家高程基准（青岛）
6. 投影变换 vs 坐标变换的区别
7. CGCS2000是地心坐标系（北斗使用），北京54/西安80是参心坐标系

#### 五、思考题 / 自测

- [x] 题：北京54与西安80坐标系有何区别？
  答：北京54用克拉索夫斯基椭球、原点在苏联；西安80用国际推荐参数、原点在西安、定向更明确
- [x] 题：为什么要进行地图投影？
  答：球面坐标不便测量计算，平面地图符合视觉习惯，但地球椭球为不可展开曲面
- [x] 题：参心坐标系与地心坐标系有何区别？
  答：参心——原点在参考椭球中心（近似地球质心）；地心——原点在地球质心，精度更高

### 第 4 章 · 空间数据的可视化表达

> 素材：[_第04章_空间数据的可视化表达.md](./_第04章_空间数据的可视化表达.md) · 主版 107 页

> 基于 OCR 全文自动提取，人工校核待补。

#### 一、核心概念（名词解释）

- [x] **数据可视化**：将数字信息转换为人可识别的图形，清晰有效传递信息，便于分析推理
- [x] **地理信息可视化三层次**：①地图数据的可视化表示 ②地理信息的可视化表示（分级+专题地图） ③空间分析结果的可视化表示
- [x] **符号化**：按实际情况确定地图符号基本形状，以颜色或形状区分事物性质
- [x] **单一符号**：大小、形状、颜色统一的符号，忽略定量差异，只反映地理位置
- [x] **分类色彩/分类符号**：不同属性值用不同符号，反映数量或质量差异
- [x] **分级色彩**：属性值按分级方法分若干级别，不同颜色表示不同级别，反映定量差异
- [x] **分级符号**：不同大小符号表示不同级别属性值，用于点/线要素
- [x] **比率符号**：属性值与符号大小一一对应，反映同级别间微小差异；数值过大时不适用
- [x] **统计符号**：饼图（整体与部分比例）、柱状图（比较/趋势）、累计柱状图（比例+趋势）
- [x] **点值符号**：用点密度表现要素空间分布，数值大区域点多
- [x] **专题地图**：突出表示一种或几种自然/社会经济现象的地图
- [x] **地图布局要素**：图名、图例、比例尺、坐标网、指北针、图框、附图

#### 二、关键公式 / 模型

- [x] **符号化方法分类**：单一符号、分类符号、分级符号、分级色彩、比率符号、组合符号、统计符号、彩色阴影
- [x] **矢量符号化**：点状→标记符号；线状→线符号；面状→填充符号
- [x] **栅格符号化**：分类栅格符号、分级栅格符号
- [x] **分级方法**：自然断点法（Jenks）、等间隔法、分位数法、标准差法

#### 三、重要案例 / 实验 / 例题

- [x] **ArcMap符号系统**：符号选择器→符号属性编辑器→样式管理器
- [x] **分类符号设置**：图层属性→符号系统→分类→设置类数和分类方法
- [x] **分级符号设置**：图层属性→符号系统→数量→分级符号
- [x] **地图输出**：布局视图→页面设置→添加图名/图例/比例尺/指北针→导出/打印

#### 四、高频考点（速记）

1. 符号化8种方法及其适用场景（必考）
2. 分级色彩 vs 分级符号的区别（颜色vs大小）
3. 比率符号的限制——属性值过大时不适用
4. 统计符号三种类型：饼图/柱状图/累计柱状图
5. 地理信息可视化三层次
6. 地图布局六大要素
7. 栅格数据符号化：分类+分级两种

#### 五、思考题 / 自测

- [x] 题：分级色彩与分级符号有何区别？
  答：分级色彩用不同颜色表示级别差异，适用于面状要素；分级符号用不同大小表示，适用于点/线要素
- [x] 题：比率符号与分级符号有何区别？
  答：比率符号一一对应（连续变化），分级符号按级别分组（离散变化）
- [x] 题：制作一幅完整专题地图需要哪些要素？
  答：图名、图例、比例尺、坐标网、指北针、图框，可能还有附图和文字说明
