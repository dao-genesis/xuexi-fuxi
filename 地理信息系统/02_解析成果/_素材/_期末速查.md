# 地理信息系统 · 期末速查表

> 一页之表，含各章关键概念、公式、考点。已吸纳 4/4 章复习要点。

| 章 | 标题 | 核心概念（≤3） | 关键公式 / 模型 | 高频考点 |
| ---: | ---- | -------------- | ---------------- | -------- |
| 1 | 地理信息系统 Geographic Information System, GIS | [x] 地理信息系统（GIS）：既是跨地球科学、空间科学、信息科学和计算机科学的一门应用基础学科，又是一项工程应用技术。以地学原理为依托，在计算机软硬件支持下，研究空间数据的采集、处理、存储、管理、分析、建模和显示。；[x] 地理数据：与地理环境要素有关的事物/现象的数量、质量、分布特征、联系和规律等的数字、文字、图像和图形总称。；[x] 地理信息：有关地理实体和地理现象的性质、特征和运动/变化状态的表征和一切有用的知识，是对地理数据的解释。 | [x] RUSLE公式（通用土壤流失方程）：A = R·K·L·S·C·P，用于土壤侵蚀量估算；[x] C-Fix模型：植被固碳量估算模型；[x] GIS功能链：数据采集与输入 → 数据编辑与更新 → 数据存储与管理 → 空间查询与分析 → 数据与图形的交互显示 → 应用模型与系统开发 | GIS的定义（必考）：跨学科应用基础+工程技术，研究空间数据采集→显示全链路；GIS五大部分构成：硬件、软件、数据、模型、人员；GIS核心功能 = 空间分析（区别于其他系统的根本标志） |
| 2 | 空间数据-知识点回顾 | [x] 空间数据采集：将地图、外业观测、航空相片、遥感图像、文本资料等转为计算机可识别的数字形式；[x] 地理空间数据：表示空间实体的位置、形状、大小及分布特征的数据；[x] 非地理空间数据（属性数据）：表示空间实体的属性特征，是对地理空间数据的说明 | [x] 栅格编码方法：直接栅格编码（最简单最直接）、游程编码、四叉树编码；[x] 矢量数据结构：点——坐标(X,Y)；线——坐标序列；面——闭合坐标序列+拓扑关系；[x] 数据模型层次：层次模型→网络模型→关系模型→对象模型 | 矢量 vs 栅格数据优缺点对比（必考）；空间数据采集方法：野外采集、地图数字化、摄影测量、遥感处理；Geodatabase三级结构：要素类→要素数据集→Geodatabase |
| 3 | 空间数据的转换与处理 | [x] 空间参考系统：确定地理目标平面位置和高程的平面坐标系和高程系的统称；[x] 地球椭球体：地球表面几何模型四类——自然表面→大地水准面→地球椭球面→数学模型；[x] 参考椭球体：与局部地区大地水准面最佳拟合的地球椭球体，需确定定位与定向 | [x] 投影变形类型：长度比 m = 实际长度/椭球面长度；面积比 p = 实际面积/椭球面面积；[x] 高斯投影分带：6°带（1:2.5万及更小比例尺）和3°带（1:1万及更大比例尺）；[x] 坐标转换方法：四参数（平面相似变换）、七参数（三维赫尔默特变换） | 四大坐标系统对比（北京54/西安80/CGCS2000/WGS84）——参心vs地心是核心区别；地球表面四层模型：自然表面→大地水准面→椭球面→数学模型；地图投影必然产生变形（长度/面积/角度） |
| 4 | 空间数据的可视化表达 | [x] 数据可视化：将数字信息转换为人可识别的图形，清晰有效传递信息，便于分析推理；[x] 地理信息可视化三层次：①地图数据的可视化表示 ②地理信息的可视化表示（分级+专题地图） ③空间分析结果的可视化表示；[x] 符号化：按实际情况确定地图符号基本形状，以颜色或形状区分事物性质 | [x] 符号化方法分类：单一符号、分类符号、分级符号、分级色彩、比率符号、组合符号、统计符号、彩色阴影；[x] 矢量符号化：点状→标记符号；线状→线符号；面状→填充符号；[x] 栅格符号化：分类栅格符号、分级栅格符号 | 符号化8种方法及其适用场景（必考）；分级色彩 vs 分级符号的区别（颜色vs大小）；比率符号的限制——属性值过大时不适用 |
