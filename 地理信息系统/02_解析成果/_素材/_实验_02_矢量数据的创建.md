# 实验二 · 矢量数据的创建

> 对应理论：第 2 章 空间数据（模型与结构）——矢量数据模型（点 / 线 / 面）。

## 一、实验目的
掌握在 **Shapefile** 与 **Geodatabase** 中创建矢量要素的方法，理解字段与坐标系的设置。

## 二、实验内容
1. 在 ArcCatalog 创建**地理数据库**（个人地理数据库 `GeoDB.mdb` 或文件地理数据库）。
2. 新建 **Shapefile**（指定要素类型 Point / Polyline / Polygon 与坐标系）。
3. 为要素类**添加字段**（Add Field：文本 / 短整型 / 长整型 / 浮点 / 双精度）。
4. 用**编辑器（Editor）**创建点 / 线 / 面要素并录入属性。
5. 导入外部数据（如由坐标表生成点要素）。

## 三、关键操作（步骤）
1. ArcCatalog 中右键文件夹 `New ▸ Shapefile`：
   - **Name** 命名；**Feature Type** 选 Point / Polyline / Polygon；
   - 点击 `Edit…` 设置 **Spatial Reference（坐标系）**（地理坐标系 GCS 或投影坐标系 PCS）。
2. 在属性表中 `Table Options ▸ Add Field` 添加字段（**注意：字段类型创建后不可更改**，需谨慎选择）。
3. `Editor ▸ Start Editing` → `Create Features` 选模板绘制要素 → 在属性表录入属性 → `Editor ▸ Stop Editing` 保存编辑。
4. 个人/文件地理数据库内同理可新建要素类（Feature Class），并可组织进要素数据集。

## 四、考点与要义
- **Shapefile vs Geodatabase**：Shapefile 由多文件（.shp/.shx/.dbf 等）组成、字段名 ≤10 字符、不支持拓扑/子类型；Geodatabase 支持要素数据集、拓扑、子类型、域、关系类，功能更强。
- **坐标系定义的重要性**：未定义坐标系的数据无法正确叠加 / 量算；创建时即应指定。
- **字段类型选择**：整型存编号、浮点/双精度存测量值、文本存名称；类型不可逆。
- **编辑会话**：必须 Start/Stop Editing 包裹，否则不能增删改要素。
