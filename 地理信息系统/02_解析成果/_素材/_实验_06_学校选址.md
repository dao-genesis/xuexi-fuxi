# 实验六 · 学校选址（栅格成本距离 + 重分类 + 加权叠加）

> 对应理论：空间分析（栅格）。教材实例 8.10.1（P323–328）。是栅格适宜性分析的经典范例。

## 一、背景与目的
为新学校选出适宜区。熟悉 ArcGIS **栅格距离制图、成本距离加权、数据重分类、多层面加权合并**等空间分析功能。

## 二、数据
`landuse`（土地利用）、`dem`（高程）、`rec_sites`（娱乐场所）、`school`（现有学校）。

## 三、选址原则与权重
| 因素 | 原则 | 权重 |
|---|---|---|
| 距娱乐设施 | 越近越好 | **0.5** |
| 距现有学校 | 越远越好（避开） | **0.25** |
| 土地利用类型 | 成本低（避开水体/湿地） | **0.125** |
| 地势（坡度） | 越平坦越好 | **0.125** |

> 权重之和 = 0.5 + 0.25 + 0.125 + 0.125 = **1.0**。

## 四、关键操作（步骤）
1. 激活 **Spatial Analyst** 扩展（`自定义 ▸ 扩展模块` 勾选）。
2. 打开 `school.mxd`；设环境：**工作空间** + **处理范围 = 与 landuse 相同** + **栅格分析像元大小 = 与 landuse 相同**（保证各层对齐可叠加）。
3. **坡度**：`Spatial Analyst ▸ 表面分析 ▸ 坡度`，输入 dem → `slope`。
4. **欧式距离**：`距离分析 ▸ 欧式距离`：`rec_sites → dis_recsites`；`school → dis_school`。
5. **重分类**（统一为 1–10 级，消除量纲）：
   - 坡度：平坦赋高值 → `reclassslope`；
   - 距娱乐场所：最近赋 10、最远赋 1 → `reclassdisr`；
   - 距现有学校：最远赋 10、最近赋 1 → `reclassdiss`；
   - 土地利用：按 Ctrl 多选删除 `water / wetland / grass`，勾选「将缺失值更改为 **NoData**」，其余赋值 → `reclassland`。
6. **加权叠加**：`Spatial Analyst ▸ 地图代数 ▸ 栅格计算器`：
   ```
   suit = reclassdisr*0.5 + reclassdiss*0.25 + reclassland*0.125 + reclassslope*0.125
   ```
   适宜性高（深色）区域即推荐选址区。

## 五、考点与要义
- **欧式距离 vs 成本距离**：欧式=直线距离；成本距离=考虑阻力表面的累积成本。
- **重分类的目的**：把不同量纲 / 范围的成本数据统一到同一可比等级，才能加权求和。
- **NoData**：表示禁止建设区（水体 / 湿地），不参与运算 / 排除在结果之外。
- **加权叠加（Weighted Overlay / 栅格计算器）**：`Σ(reclass × weight)`，权重反映各因素重要性。
- **环境设置（像元大小 / 范围对齐）**是栅格多层运算正确叠加的前提。
