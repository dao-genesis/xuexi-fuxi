# 实验五 · 市区择房分析（缓冲区分析 + 叠置分析）

> 对应理论：空间分析（矢量）。教材实例 7.5.1（P264–270）。综合运用**缓冲区分析**与**叠置分析**。

## 一、背景与目的
为购房者在城区中选出「最佳购房区」：既要远离噪声、又要靠近商业中心 / 名校 / 名胜。核心是**缓冲区（Buffer）+ 叠置（Overlay）**。

## 二、数据
`network`（交通网络）、`marketplace`（商业中心）、`school`（名牌高中）、`famous place`（名胜古迹）、`city.mxd`。

## 三、四个择房条件
| 条件 | 含义 | 实现 |
|---|---|---|
| ① 远离主干道噪声 | 距主干道 **200m 之外** | network 选主干道后做 200m 缓冲，再**擦除** |
| ② 靠近大型商业中心 | 位于商业中心影响范围内 | marketplace 缓冲，半径=属性字段 **YUZHI** |
| ③ 靠近名牌高中 | 距名校 **750m 之内** | school 缓冲 750m |
| ④ 靠近名胜古迹 | 距名胜 **500m 之内** | famous place 缓冲 500m |

## 四、关键操作（步骤）
1. **主干道噪声缓冲区**：对 `network` 按属性选择 `"TYPE" = 'ST'` 选出主要道路（街道）→ 用 Buffer，仅对所选要素分析，半径 **200m** → 得噪声缓冲区。
2. **商业中心影响范围**：对 `marketplace` 做缓冲，缓冲半径取**属性字段 YUZHI**（不同中心影响范围不同）。
3. **名牌高中影响范围**：对 `school` 做缓冲，半径 **750m**。
4. **名胜古迹影响范围**：对 `famous place` 做缓冲，半径 **500m**。
5. **叠置分析求最佳区**：
   - 对商业中心、名校、名胜三缓冲区做【分析工具 ▸ 叠加分析 ▸ **相交 Intersect**】→ 同时满足 3 个条件的区域；
   - 再用主干道噪声缓冲区对上一步结果做【**擦除 Erase**】（排除噪声区）→ **同时满足 4 个条件的最佳购房区**。

## 五、考点与要义
- **缓冲区建立三要素**：缓冲对象图层、缓冲半径（定值或字段值）、边界设定类型（是否融合 Dissolve）。
- **按属性选择 SQL**：`"TYPE"='ST'`，配合「仅对所选要素分析」。
- **叠置分析语义**：`Intersect` 求交（取同时满足）；`Erase` 擦除（剔除某条件区）；`Union` 求并；`Clip` 裁剪。
- **思路模型**：正向条件用相交叠加，负向条件（远离）用缓冲后擦除——这是选址 / 适宜性分析的通用套路。
