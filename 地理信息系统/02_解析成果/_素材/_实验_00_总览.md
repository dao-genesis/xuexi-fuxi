# 实验上机 · 总览（理论 ↔ 实验 · 融为一体）

> 本课程《地理信息系统》= **理论**（4 章：概论 / 空间数据 / 转换与处理 / 可视化表达）+ **实验**（8 个 ArcGIS 上机实验）。
> 实验指导书来源：新疆大学 生态与环境学院 · 地理信息系统课程（ArcGIS 平台，黄源生 编）。
> 下面把 8 个实验逐一精讲（实验目的 / 内容 / 步骤 / 关键操作 / 考点 / 易错），并与理论各章一一对应，做到**学—练—考**闭环。

## 二、ArcGIS 三大组件（贯穿全部实验）

- **ArcMap**：地图显示、编辑、符号化、制图输出、空间分析（含 ArcToolbox 与扩展模块）。数据视图 / 布局视图两种显示模式。
- **ArcCatalog**：数据管理（浏览、创建、组织 Shapefile / Geodatabase / 要素类 / 要素数据集 / 拓扑），连接文件夹。
- **ArcToolbox**：地理处理工具箱（分析工具、转换工具、Spatial Analyst、数据管理等）。

## 三、矢量 vs 栅格实验主线（一眼看清方法谱系）

- **矢量上机线**：实验二（创建）→ 实验三（编辑 / 拓扑 / 坐标系）→ 实验四（专题制图）→ 实验五（缓冲区 + 叠置）。
- **栅格上机线**：实验六（成本距离 + 重分类 + 加权叠加）→ 实验七（插值 + 分区统计 + 栅格计算）→ 实验八（水文分析 + ModelBuilder）。
- 两条线对应理论中「矢量数据结构」「栅格数据结构」「空间分析」的实操落地。

## 四、上机高频考点速记（实验部分）

1. **缓冲区分析**：对点 / 线 / 面按距离生成影响范围；半径可为定值或取自属性字段（如实验五 YUZHI）。
2. **叠置分析**：`Intersect`（交集，求同时满足）、`Union`（并集）、`Erase`（擦除，排除某条件）、`Clip`（裁剪）。
3. **拓扑规则**：Must Not Have Dangles（悬挂）、Must Not Have Pseudo Nodes（伪节点）、Must Not Intersect（相交）；错误类型：缝隙 / 叠盖 / 悬挂 / 伪节点。
4. **欧式距离 vs 成本距离**：前者直线距离，后者考虑成本表面（阻力）累积距离。
5. **重分类**：把连续 / 多值数据按等间距等规则归并为统一等级（如 1–10），统一量纲后才能加权叠加；NoData 表示禁止 / 排除区。
6. **加权叠加**：`suit = Σ(reclass_i × weight_i)`，权重之和=1（实验六：0.5+0.25+0.125+0.125）。
7. **空间插值（Kriging）**：由离散点推求连续表面；普通 / 泛克里金、半变异函数模型；掩膜（Mask）限定插值范围。
8. **分区统计**：Zonal Statistics（输出栅格）/ Zonal Statistics as Table（输出表）；Zone field 定义分区；统计量 MEAN / SUM / MAX 等。
9. **水文分析链**：Fill（填洼）→ Flow Direction（流向）→ Flow Accumulation（汇流累积）→ 阈值提取河网 → Stream Link / Stream to Feature → Watershed（流域）→ Raster to Polygon。
10. **ModelBuilder**：可视化串联地理处理工具的自动化建模；模型只能存于 Toolbox（不能直接存文件夹）。
