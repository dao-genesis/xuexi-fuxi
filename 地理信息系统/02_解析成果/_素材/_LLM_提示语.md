# 地理信息系统 · LLM 喂入提示语

> 复习占位（核心概念/公式/考点等）可由具备视觉能力的 LLM 协填。下列提示语可直接复制使用。

## 提示语 一 · 按章填充素材

```
你将以图文方式阅读《地理信息系统》（黄源生 教）某章之课件主版。
请按下列结构整理：

1. 核心概念（名词解释，每条 1-3 句，标出页码）
2. 关键公式 / 模型（含适用条件与典型例题）
3. 重要案例 / 实验（含原理、步骤、结论）
4. 高频考点（按 必考/常考/可能考 三档分级）
5. 易错 / 易混点（成对列出）
6. 三道思考题 + 参考答案
7. 与前后章之关联（一句话）

要求：
- 用 markdown，与素材 md 占位结构对齐
- 每条不超 5 行，必要时分点
- 保留原图序号引用（如 参 p012）
```

## 提示语 二 · 生成速查表

```
已读《地理信息系统》全部 4 章。
请生成一份紧凑速查表（一页，markdown 表格），每章一行：

| 章 | 标题 | 核心概念≤3 | 关键公式 | 高频考点 |

再补：
- 易错点 10 条（错过即列，标章·页）
- 名词速对 20 条（术语→释义→章）
- 章际关联图（mermaid 图，节点为章，边为概念衔接）
```

## 提示语 三 · 期末模拟卷

```
以《地理信息系统》全部章节为蓝本，出一份期末模拟卷：

- 名词解释 5（覆盖不同章）
- 简答题 5（综合 2-3 章之联系）
- 论述题 2（深考主线）
- 计算 / 应用题 2（含步骤）
- 全卷 100 分，2 小时
- 附详细答案与评分标准

出题原则：
- 不偏不怪，覆盖主版课件之高频内容
- 体现各章之相对权重（页数 / 题量 大致正比）
```
