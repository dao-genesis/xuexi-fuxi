> 综合：各高校考研真题、自考真题、期末试题高频题

## 一 · 名词解释（高频30题）

### 1. 地理信息系统（GIS）
在计算机软硬件支持下，对空间数据进行采集、存储、管理、分析、显示和输出的信息系统。它既是跨学科的应用基础学科，又是一项工程技术。

### 2. 空间数据
描述空间实体的位置、形状、大小及分布特征的数据，具有空间特征、属性特征和时序特征三大基本特征。

### 3. 地理信息
有关地理实体和地理现象的性质、特征和运动/变化状态的表征和一切有用的知识，是对地理数据的解释。

### 4. 拓扑关系
空间实体间的相邻、关联、包含关系。拓扑关系不随地图投影和比例尺变化而改变，是空间分析的基础。

### 5. 游程编码（RLE）
一种栅格数据压缩方法，记录属性值及其连续出现的个数。适合大片同值区域的压缩，压缩效果取决于数据的均匀程度。

### 6. 四叉树编码
递归四等分的层次性数据结构，同值区合并为一个节点，不同值区继续细分。适合具有层次性和区域一致性的数据。

### 7. 数字高程模型（DEM）
通过有限地形高程数据实现对地面高程的数字化模拟，是地形分析的数学基础。表达形式有规则格网（Grid）和不规则三角网（TIN）。

### 8. 不规则三角网（TIN）
由不规则分布的数据点按Delaunay三角准则自动连接形成的三角网络，能根据地形的起伏特征改变采样密度。

### 9. 元数据
描述数据的数据，说明数据的内容、质量、状况和其他有关特征的信息。用于数据共享、查询和管理。

### 10. 地图投影
将地球椭球面上的点映射到平面上的数学方法。实质是建立地球椭球面与平面之间点的一一对应关系。

### 11. 高斯-克吕格投影
等角横切椭圆柱投影。中央经线无变形，离中央经线越远变形越大。我国基本比例尺地形图采用此投影，6°分带（13-23带）和3°分带。

### 12. 仿射变换
6参数坐标变换方法，可在X和Y方向上进行不同比例的缩放，同时进行扭曲、旋转和平移。特性：直线变换后仍为直线，平行线仍为平行线。

### 13. 缓冲区分析
对空间实体周围建立一定宽度的缓冲区多边形，用于分析实体对周围环境的影响范围。

### 14. 叠置分析
将两层或多层空间数据叠加，产生新的空间特征和属性的分析方法。是GIS最基本的空间分析功能之一。

### 15. 空间内插
通过已知点数据推求未知点数据的数学方法。常用方法有距离反比加权法（IDW）、克里金法（Kriging）、样条函数法。

### 16. 克里金插值（Kriging）
基于区域化变量理论，利用变异函数，对未知点进行最优线性无偏估计的空间插值方法。

### 17. 空间索引
基于空间实体位置和形状或空间关系，按一定顺序排列的数据结构。常见类型有范围索引、格网索引、四叉树索引。

### 18. Geodatabase
ESRI提出的面向对象的空间数据存储模型，三级结构：要素类→要素数据集→Geodatabase。

### 19. WebGIS
基于Internet/Intranet的地理信息系统，利用Web技术实现空间数据的发布、共享和互操作。

### 20. 空间数据引擎（SDE）
中间件技术，实现空间数据在关系数据库中的存储管理，使GIS应用与数据库之间建立连接桥梁。

### 21. 数据可视化
将数字信息转换为人可识别的图形图像，清晰有效地传递信息的过程。

### 22. 符号化
按实际情况确定地图符号的基本形状，以颜色或形状区分事物性质，将空间数据转换为地图表达的过程。

### 23. 专题地图
突出表示一种或几种主题要素的地图，如人口分布图、土地利用图、等温线图等。

### 24. 大地水准面
假设平均海水面延伸穿过大陆形成的封闭曲面，是高程的基准面，也是地球形状的物理表面。

### 25. 参考椭球体
与局部地区大地水准面最佳拟合的地球椭球体，需确定定位与定向参数。

### 26. CGCS2000
中国大地坐标系2000，原点在地球质心的地心坐标系，椭球体为CGCS2000椭球，是我国新一代国家大地坐标系。

### 27. Dijkstra算法
求解图中单源最短路径的经典算法，采用贪心策略逐步扩展最短路径树。时间复杂度O(n²)。

### 28. 泰森多边形（Voronoi图）
由一组点生成的多边形，每个多边形内的点到其生成点的距离小于到其他任何生成点的距离。与Delaunay三角网互为对偶。

### 29. 制图综合
根据地图用途和比例尺，对地图内容进行选取、概括、合并和移位，以保持地图清晰可读的过程。

### 30. 3S集成
GPS（定位）、RS（遥感获取）、GIS（分析管理）三种技术的集成应用，实现从数据获取到分析应用的全流程。

## 二 · 简答题（高频20题）

### 1. 简述GIS的基本组成

GIS由五部分组成：
1. **硬件系统**：服务器、工作站、输入设备（数字化仪、扫描仪、GPS）、输出设备（绘图仪、打印机）、存储设备
2. **软件系统**：系统软件（OS）、GIS基础软件（ArcGIS、MapInfo、QGIS）、应用软件
3. **数据**：空间数据+属性数据，是GIS的核心内容
4. **模型**：空间分析模型、应用模型
5. **人员**：系统开发人员、数据生产者、最终用户

### 2. 简述GIS的基本功能

1. **数据采集与输入**：将各种来源数据转为计算机可识别形式
2. **数据编辑与更新**：数据检查、修改、维护
3. **数据存储与管理**：空间数据库的组织与管理
4. **空间查询与分析**：GIS核心功能，区别于其他系统的根本标志
5. **数据与图形的交互显示**：可视化表达
6. **应用模型与系统开发**：面向具体应用的模型构建

### 3. 简述矢量数据与栅格数据的区别（必考）

| 方面 | 矢量 | 栅格 |
|------|------|------|
| 数据结构 | 复杂、精确 | 简单、规则 |
| 数据量 | 小 | 大 |
| 拓扑关系 | 易建立 | 难建立 |
| 图形精度 | 高 | 取决于分辨率 |
| 叠置分析 | 复杂 | 简单 |
| 图形显示 | 精美 | 有锯齿 |
| 适宜对象 | 离散对象 | 连续场数据 |

### 4. 简述空间数据的三大基本特征

1. **空间特征**：描述实体位置、形状、大小及实体间的空间关系（拓扑、方向、度量），是GIS区别于其他信息系统的根本标志
2. **属性特征**：描述实体的性质和状态，如类型、等级、数量等
3. **时序特征**：描述实体随时间变化的特征，反映数据的现势性

### 5. 简述栅格数据编码方法

1. **直接栅格编码**：最简单，逐行逐列记录像元值
2. **游程编码（RLE）**：记录属性值+连续个数，适合大片同值区域
3. **四叉树编码**：递归四等分，同值区合并，适合层次性数据
4. **链码（Freeman码）**：用方向码记录边界，适合线状要素

### 6. 简述高斯-克吕格投影的特点

1. 中央经线上没有任何变形，投影后保持长度不变
2. 除中央经线外，其他任何点长度比大于1
3. 同一纬线上，离中央经线越远变形越大
4. 同一经线上，纬度越低变形越大
5. 属于等角投影，面积比为长度比的平方
6. 长度比等变形线平行于中央经线
7. 分带控制变形：6°带和3°带

### 7. 简述仿射变换的特性

仿射变换是6参数变换，特性：
1. 直线变换后仍为直线
2. 平行线变换后仍为平行线
3. 不同方向上的长度比发生变化
4. 可在X和Y方向上进行不同比例的缩放
5. 同时进行扭曲、旋转和平移

### 8. 简述空间数据处理的三方面内容

1. **数据变换**：数据从一种数学状态到另一种数学状态的变换，包括几何纠正和地图投影转换
2. **数据重构**：数据从一种格式到另一种格式的转换，包括结构转换、格式转换、类型替换
3. **数据提取**：对数据进行某种条件的取舍，包括类型提取、窗口提取、空间内插

### 9. 简述空间内插的三种类型

1. **分块内插法**：将区域分块，每块内用简单函数拟合。包括线性内插、双线性多项式内插、二元样条函数内插
2. **逐点内插法**：以待插点为中心，利用周围一定范围内的已知点进行内插。包括移动拟合法、距离反比加权法（IDW）、克里金法
3. **整体内插法**：在整个区域上用统一函数拟合。包括N次多项式拟合法

### 10. 简述投影转换的四种方式

1. **正解变换**：直接建立两个投影间的解析关系式
2. **反解变换**：先反解出地理坐标，再代入新投影公式
3. **数值变换**：原投影公式未知时，利用离散点坐标数值逼近建立关系式
4. **解析-数值变换**：新投影已知、原投影未知，先反解出地理坐标再代入新投影

### 11. 简述GIS与MIS、CAD的区别

| 方面 | GIS | MIS | CAD |
|------|-----|-----|-----|
| 数据类型 | 空间+属性 | 属性为主 | 图形为主 |
| 坐标系 | 地理坐标系 | 无 | 局部坐标系 |
| 拓扑关系 | 必须建立 | 无 | 一般不建立 |
| 核心功能 | 空间分析 | 统计分析 | 制图设计 |
| 数据量 | 大 | 中 | 小 |

### 12. 简述空间数据质量的要素

1. **位置精度**：空间位置的准确程度
2. **属性精度**：属性值的准确程度
3. **逻辑一致性**：数据间逻辑关系的正确性
4. **完整性**：数据是否完整无遗漏
5. **时间精度**：数据的时间现势性
6. **数据世系**：数据来源和处理历史

### 13. 简述空间数据采集的主要方法

1. **野外采集**：GPS测量、全站仪等
2. **地图数字化**：手扶跟踪数字化、扫描矢量化
3. **摄影测量**：航空摄影、数字摄影测量
4. **遥感处理**：卫星遥感影像解译与分类

### 14. 简述缓冲区分析的步骤

1. 确定缓冲目标（点、线、面）
2. 设定缓冲距离
3. 生成缓冲区多边形
4. 进行叠加分析
5. 提取结果

### 15. 简述叠置分析的三种类型

1. **点与多边形叠置**：判断点落在哪个多边形内，获得多边形属性
2. **线与多边形叠置**：判断线穿越哪些多边形，将线段分割并赋予多边形属性
3. **多边形与多边形叠置**：将两个多边形图层叠加产生新多边形，属性合并

### 16. 简述符号化的8种方法

1. **单一符号**：所有要素同一符号
2. **分类符号**：按属性类别不同符号
3. **分级符号**：按数值大小不同大小符号
4. **分级色彩**：按数值大小不同颜色
5. **比率符号**：符号大小与属性值成比例
6. **组合符号**：多属性组合表示
7. **统计符号**：饼图/柱状图符号
8. **彩色阴影**：色彩深浅表示

### 17. 简述关系数据库的三类完整性

1. **实体完整性**：主键非空原则
2. **参照完整性**：不允许引用不存在的实体
3. **用户自定义完整性**：具体应用所涉及数据必须满足的语义要求

### 18. 简述数据库范式

1. **1NF**：每个属性必须是原子、不可分割的数据项
2. **2NF**：在1NF基础上，每个非主属性完全函数依赖关键字
3. **3NF**：在2NF基础上，消除传递函数依赖
4. **BCNF**：每个决定因素都是候选关键字

### 19. 简述常见的空间索引方法

1. **范围索引**：记录每个空间实体的最大最小坐标
2. **格网索引**：将区域划分为大小相等的网格，记录每个网格内空间实体的地址
3. **四叉树索引**：递归四等分区域，用莫顿码记录实体位置

### 20. 简述GIS的发展趋势

1. WebGIS和云GIS
2. 移动GIS和嵌入式GIS
3. 三维GIS和时态GIS
4. GIS与大数据、AI融合
5. 互操作和标准化
6. 开源GIS发展

## 三 · 论述题（高频10题）

### 1. 论述GIS空间分析功能及其在地学中的应用

**空间分析功能：**
1. **空间查询**：属性查图形、图形查属性
2. **叠置分析**：多图层叠加产生新特征
3. **缓冲区分析**：分析实体影响范围
4. **网络分析**：最短路径、最佳路径、资源分配
5. **地形分析**：坡度、坡向、等高线生成
6. **空间统计分析**：空间自相关、空间插值

**地学应用：**
- 城市规划：用地适宜性评价
- 环境保护：污染源影响范围分析
- 交通规划：最短路径选择
- 农业：作物适宜性评价
- 地质：矿产资源评价
- 防灾减灾：洪水淹没分析

### 2. 论述四种坐标系统的区别与联系

| 坐标系 | 类型 | 原点 | 椭球体 | 特点 |
|--------|------|------|--------|------|
| 北京54 | 参心 | 参考椭球中心 | 克拉索夫斯基 | 原苏联体系，精度较低 |
| 西安80 | 参心 | 参考椭球中心 | IUGG1975 | 中国自主建立 |
| CGCS2000 | 地心 | 地球质心 | CGCS2000椭球 | 中国新一代地心坐标系 |
| WGS84 | 地心 | 地球质心 | WGS84椭球 | GPS使用，全球统一 |

**联系：**
- 参心坐标系与地心坐标系的区别在于原点位置不同
- 参心坐标系适合局部地区使用，地心坐标系适合全球应用
- 坐标转换需要通过七参数或四参数变换实现
- CGCS2000与WGS84基本一致，差异在厘米级

### 3. 论述GIS在数字城市中的应用

1. **城市规划管理**：用地审批、建筑管理、规划方案评估
2. **市政设施管理**：地下管网、供水排水、电力通信
3. **交通管理**：路网规划、交通流量分析、应急调度
4. **环境保护**：污染源监测、环境质量评价
5. **公共安全**：消防调度、犯罪分析、应急指挥
6. **土地管理**：地籍管理、土地利用变更监测
7. **社区服务**：公共服务设施选址、社区资源分配

### 4. 论述地图投影的选择原则

1. **根据制图区域范围**：
   - 世界图：多圆柱投影、摩尔威德投影
   - 半球图：方位投影
   - 中纬度地区：圆锥投影（兰伯特、阿尔伯斯）
   - 赤道地区：圆柱投影（墨卡托）
2. **根据制图目的**：
   - 航海图：等角投影（墨卡托）
   - 面积统计：等面积投影（阿尔伯斯）
   - 地形图：等角投影（高斯-克吕格）
3. **根据变形分布**：
   - 控制角度变形→等角投影
   - 控制面积变形→等面积投影
   - 均衡变形→任意投影

### 5. 论述GIS项目系统设计的步骤

1. **需求分析**：明确系统目标、功能需求、数据需求
2. **可行性分析**：技术可行性、经济可行性、社会可行性
3. **系统总体设计**：
   - 逻辑设计：数据流程、功能模块
   - 物理设计：硬件配置、软件选择、网络架构
4. **数据库设计**：
   - 概念设计：E-R图
   - 逻辑设计：关系模型
   - 物理设计：存储结构、索引
5. **详细设计**：模块接口、算法、编码规范
6. **系统实现**：编码、测试、数据入库
7. **系统运行与维护**：日常管理、更新升级

### 6. 论述3S集成技术及其应用

**3S含义：**
- GPS：全球定位系统，提供实时定位信息
- RS：遥感，提供大范围空间数据获取
- GIS：地理信息系统，提供空间数据管理和分析

**集成方式：**
1. GPS+GIS：实时定位与导航
2. RS+GIS：遥感数据处理与分析
3. GPS+RS：遥感影像几何校正
4. GPS+RS+GIS：全流程集成

**应用：**
- 精准农业：GPS定位+RS监测+GIS分析
- 智慧交通：GPS车辆定位+GIS路径规划
- 防灾减灾：RS灾害监测+GIS评估+GPS定位
- 国土资源调查：RS数据获取+GIS管理+GPS验证

### 7. 论述空间数据不确定性及处理方法

**不确定性来源：**
1. 数据源不确定性：测量误差、分类误差
2. 数据处理不确定性：坐标转换误差、内插误差
3. 数据使用不确定性：尺度效应、语义模糊

**处理方法：**
1. 误差传播分析：Monte Carlo模拟
2. 数据质量评价：位置精度、属性精度评估
3. 元数据管理：记录数据来源和处理过程
4. 不确定性可视化：误差带、概率等值线
5. 敏感性分析：评估不确定性对结果的影响

### 8. 论述GIS互操作及其解决方法

**互操作问题：**
1. 数据格式不兼容
2. 坐标参考系不一致
3. 语义差异
4. 软件平台不同

**解决方法：**
1. **数据标准**：制定统一的数据标准和规范
2. **开放地理数据互操作规范（OGC）**：WMS、WFS、WCS等服务
3. **数据转换**：使用中间格式（如Shapefile、GML）
4. **Web服务**：基于SOA架构的服务共享
5. **语义互操作**：本体论（Ontology）方法

## 四 · 计算题（典型5题）

### 1. Morton码计算

**题目**：在8×8栅格矩阵中，求第6行第5列（行号从0开始）的Morton码。

**解法**：
- 行号6 = 110（二进制）
- 列号5 = 101（二进制）
- 交错排列：1**1**1**0**0**1** → 111001 = 57

> 注意：另一种常见编号从1开始，第6行第5列=行5列4
> 行5=101，列4=100，交错=1**1**0**0**1**0**=110010=50

### 2. 仿射变换参数计算

**题目**：已知4个控制点的旧坐标和新坐标，求仿射变换6参数。

**公式**：
```
X' = a₀ + a₁X + a₂Y
Y' = b₀ + b₁X + b₂Y
```

**步骤**：
1. 列出方程组（至少3个点）
2. 用最小二乘法求解6个参数
3. 用第4个点验证精度

### 3. 游程编码

**题目**：对以下栅格数据进行游程编码：
```
A A A B B
A A B B B
C C C C C
C C D D D
```

**解法**：
- 第1行：(A,3)(B,2)
- 第2行：(A,2)(B,3)
- 第3行：(C,5)
- 第4行：(C,2)(D,3)

### 4. 缓冲区面积计算

**题目**：一条长L=1000m的线状要素，缓冲距离d=50m，求缓冲区面积。

**解法**：
- 矩形部分：L × 2d = 1000 × 100 = 100,000 m²
- 两端半圆：π × d² = π × 2500 ≈ 7,854 m²
- 总面积 ≈ 107,854 m²

### 5. 高斯-克吕格投影带号计算

**题目**：已知某地经度为117°E，求其在6°分带中的带号和中央经线。

**解法**：
- 6°分带：带号 = ⌈117/6⌉ = 20（从0°经线算起）
- 中央经线 = 20 × 6 - 3 = 117°E
- 验证：117°恰好在第20带中央经线上

> 3°分带：带号 = ⌈117/3⌉ = 39，中央经线 = 39 × 3 = 117°E

## 五 · 综合应用题（典型3题）

### 1. 洪水淹没损失预测分析

**数据源**：DEM、土地利用图、政区图、道路交通图、人口分布图、遥感影像、社会经济统计年鉴

**步骤**：
1. 利用DEM提取地形特征（坡度、坡向、汇水累积量）
2. 根据水文模型确定洪水淹没范围
3. 将淹没范围与土地利用图叠置，获取淹没区用地类型
4. 与政区图叠置，获取各行政单元淹没面积
5. 与人口分布图叠置，估算受灾人口
6. 结合社会经济统计年鉴，估算经济损失
7. 利用道路交通图评估交通影响
8. 生成洪水淹没损失评估报告和专题地图

### 2. 城市公交站点选址

**步骤**：
1. 收集数据：道路网、居民区分布、人口密度、现有站点、学校医院等
2. 建立居民区缓冲区（步行500m范围）
3. 对现有站点做缓冲区分析，找出服务盲区
4. 将盲区与人口密度图叠置，确定优先选址区域
5. 结合道路网进行可达性分析
6. 用AHP法综合评价候选站点
7. 生成选址方案专题图

### 3. 土地利用变化检测

**步骤**：
1. 获取两期遥感影像
2. 几何校正和配准
3. 影像分类获取两期土地利用图
4. 将两期土地利用图叠置分析
5. 计算转移矩阵（各类间转化面积）
6. 分析变化趋势和驱动力
7. 生成变化检测专题图
