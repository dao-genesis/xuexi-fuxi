## 一、核心定义（必考·名词解释）

### 1.1 地理信息系统 GIS

> **GIS** = Geographic Information System
> 既是跨**地球科学、空间科学、信息科学和计算机科学**的一门**应用基础学科**，又是一项**工程应用技术**。
> 以地学原理为依托，在计算机软硬件支持下，研究空间数据的**采集、处理、存储、管理、分析、建模和显示**。

**答题要点**：①跨学科性质 ②计算机软硬件支持 ③空间数据全链路（采集→显示）

### 1.2 数据 vs 信息 vs 地理数据 vs 地理信息

| 概念 | 定义 | 关键词 |
|------|------|--------|
| **数据** | 未经加工的原始事实记录 | 原始、载体 |
| **信息** | 用数据表示事物内容/数量/特征，向人们提供关于现实世界新的事实和知识 | 解释、知识 |
| **地理数据** | 与地理环境要素有关的事物/现象的**数量、质量、分布特征、联系和规律**等的数字、文字、图像和图形总称 | 数量+质量+分布 |
| **地理信息** | 有关地理实体和地理现象的**性质、特征和运动/变化状态**的表征和一切有用的知识 | 性质+变化+解释 |

**核心关系**：信息来源于数据 → 信息是数据的表达 → 数据是信息的载体 → 地理信息是对地理数据的解释

### 1.3 地理数据五大特征

| 特征 | 含义 |
|------|------|
| **空间分布性** | 地理对象在空间上有确定位置和分布模式 |
| **时间序列性** | 地理现象随时间动态变化 |
| **数量海量性** | 空间对象本身数据量大，分析过程产生更多数据 |
| **载体多样性** | 图形、图像、文字、表格等多种形式 |
| **位置-属性对应性** | 空间位置与属性紧密关联，这是GIS区别于MIS的本质 |

### 1.4 地理信息特有特征

> 地理信息具备信息基本特征（客观性、适用性、可传输性、共享性），此外还有独特特性：

| 特征 | 说明 |
|------|------|
| **空间相关性** | 近邻事物关联更强（Tobler地理学第一定律） |
| **空间层次性** | 不同尺度/分辨率下表现不同 |
| **空间区域性** | 同一现象在不同区域特征不同 |
| **空间多样性** | 同一区域多种现象共存 |

## 二、GIS的构成（五大部分·必考简答）

```
        ┌─────────┐
        │ 应用人员 │  ← 系统开发/数据生产/终端用户
        ├─────────┤
        │ 地学模型 │  ← 针对具体地学问题的数学模型
        ├─────────┤
        │ 空间数据 │  ← GIS核心内容（矢量+栅格+属性）
        ├─────────┤
        │ 软件系统 │  ← 支撑→平台→应用（三层）
        ├─────────┤
        │ 硬件系统 │  ← 输入→处理→输出+存储
        └─────────┘
```

| 组成 | 详细说明 | 考试关键词 |
|------|----------|-----------|
| **硬件系统** | 输入设备（数字化仪、扫描仪、GPS）→ 处理设备（服务器、工作站）→ 输出设备（绘图仪、打印机）→ 存储设备 | 物理外壳 |
| **软件系统** | ①支撑软件（OS/DB）②平台软件（ArcGIS/QGIS/MapInfo）③应用软件（二次开发） | 三层结构 |
| **空间数据** | 矢量+栅格+属性；几何坐标+空间关系+非几何属性（定性+定量）；**数据库建设占系统投资70%** | 核心内容 |
| **地学模型** | ①基于GIS已有操作构建 ②无已有工具时编程构建 | 应用深化 |
| **应用人员** | 系统开发、项目管理、科学研究、软件设计、数据维护、普通用户 | 不可或缺 |

## 三、GIS基本特征

1. **数据的空间定位特征**：GIS = 空间数据 + 属性数据的综合体（本质特征）
2. **空间关系处理的复杂性**：需同时处理图像、空间关系、属性数据
3. **海量数据管理的能力**：空间对象本身海量 + 分析过程产生海量数据

> ⚡ **核心论断**：GIS的核心功能 = **空间分析** —— 这是GIS与其他计算机系统的**根本区别**

## 四、GIS功能链（必考·简答）

```
数据采集与输入 → 数据编辑与更新 → 数据存储与管理 → 空间查询与分析 → 数据与图形交互显示 → 应用模型与系统开发
```

| 功能 | 核心内容 | 考试要点 |
|------|----------|----------|
| **数据采集与输入** | 将地图/遥感/外业/文本转为计算机可识别数字形式 | 数据库建设占投资**70%**；重要研究内容：数据共享和自动化输入 |
| **数据编辑与更新** | 图形编辑（拓扑/几何/图幅拼接/投影变换/误差校正）+ 属性编辑 | 空间实体动态发展，数据只反映瞬时特征 |
| **数据存储与管理** | 空间数据库：矢量/栅格/混合模型组织；空间数据+属性数据融为一体 | 消除"数据孤岛" |
| **空间查询与分析** | ①空间查询（最基本功能）②空间分析（**核心功能**）③模型分析（应用深化标志） | **根本区别** |
| **数据与图形交互显示** | 屏幕显示 + 硬拷贝（报告/表格/地图）；强调**地图输出**功能：全要素地图/专题图/统计图 | 可视化 |
| **应用模型与系统开发** | 二次开发功能 + 完整API和开发环境 | 面向行业需求 |

## 六、GIS与其他系统区别（简答高频）

| 对比维度 | GIS | CAD | MIS |
|----------|-----|-----|-----|
| **数据类型** | 空间+属性 | 图形为主 | 属性为主 |
| **坐标系** | 地理坐标系 | 局部坐标系 | 无 |
| **拓扑关系** | 必须建立 | 一般不建立 | 无 |
| **核心功能** | 空间分析 | 制图设计 | 统计分析 |
| **数据量** | 大 | 中 | 小 |

> 答题技巧：从数据类型、坐标系、拓扑关系、核心功能四个维度对比

| 模块 | 功能 | 记忆口诀 |

## 九、环境GIS应用

### 9.1 三种应用模式

1. **环境科学研究工具**：辅助手段
2. **环境管理**：长期、稳定、高效运行的业务化服务系统
3. **公众服务**

### 9.2 十大应用需求

> 污染控制 · 环境监测 · 建设项目管理 · 环境规划 · 污染源远程监控

1. **环境空间数据处理功能**：调整数据结构，灵活组织数据内容
2. **环境空间数据分析功能**：从已有数据中获得规律性知识
3. **环境空间数据维护功能**：图层/图形编辑/数据访问
4. **环境应用功能**：动态查询显示/专题地图/环境质量评价

## 十一、经典试题精选

### 名词解释

1. **地理信息系统（GIS）**：在计算机软硬件支持下，对空间数据进行采集、存储、管理、分析、显示和输出的信息系统。既是跨学科应用基础学科，又是一项工程技术。
2. **地理数据**：与地理环境要素有关的事物/现象的数量、质量、分布特征、联系和规律的数字、文字、图像和图形总称。
3. **地理信息**：有关地理实体和地理现象的性质、特征和运动/变化状态的表征和一切有用的知识，是对地理数据的解释。
4. **3S集成**：GPS（定位）+ RS（遥感获取）+ GIS（分析管理）三种技术的集成应用。

### 简答题

**Q1：简述GIS的基本组成**
> GIS由五部分组成：①硬件系统（服务器/工作站/输入输出设备/存储设备）②软件系统（支撑软件/平台软件/应用软件）③数据（空间数据+属性数据，GIS核心）④模型（空间分析模型/应用模型）⑤人员（开发人员/数据生产者/最终用户）

**Q2：简述GIS的基本功能**
> ①数据采集与输入 ②数据编辑与更新 ③数据存储与管理 ④空间查询与分析（核心功能）⑤数据与图形的交互显示 ⑥应用模型与系统开发

**Q3：GIS与MIS、CAD的区别**
> 见第六节对比表。核心区别：GIS处理空间+属性数据、使用地理坐标系、必须建立拓扑关系、核心功能是空间分析。

### 论述题

**Q：论述GIS空间分析功能及其在地学中的应用**
> 空间分析功能：①空间查询（属性查图形/图形查属性）②叠置分析（多图层叠加产生新特征）③缓冲区分析（分析实体影响范围）④网络分析（最短/最佳路径）⑤地形分析（坡度/坡向/等高线）⑥空间统计分析（空间自相关/空间插值）
> 地学应用：城市规划（用地适宜性评价）、环境管理（污染扩散模拟）、资源管理（矿产分布制图）、灾害评估（洪水淹没分析）、交通规划（路径优化）
