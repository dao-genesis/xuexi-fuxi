## 第1章 · 地理信息系统基础

### 1.1 核心定义

| 术语 | 释义 |
|------|------|
| 地理信息系统（GIS） | 在计算机软硬件支持下，对空间数据进行采集、存储、管理、分析、显示和输出的信息系统。既是跨学科应用基础学科，又是一项工程技术 |
| 地理数据 | 与地理环境要素有关的事物/现象的数量、质量、分布特征、联系和规律的数字、文字、图像和图形总称 |
| 地理信息 | 有关地理实体和地理现象的性质、特征和运动/变化状态的表征和一切有用的知识，是对地理数据的解释 |
| 地理信息特征 | 空间特征、属性特征、时序特征 |
| 信息 | 客观事物运动状态的表征，具有客观性、传输性、共享性、适用性 |
| 数据 | 未加工的原始事实，是信息的载体 |

### 1.2 GIS 组成与功能

| 组成部分 | 说明 |
|----------|------|
| 硬件系统 | 服务器、工作站、输入设备（数字化仪、扫描仪、GPS）、输出设备（绘图仪、打印机）、存储设备 |
| 软件系统 | 系统软件（OS）、GIS基础软件（ArcGIS、MapInfo、QGIS）、应用软件 |
| 数据 | 空间数据+属性数据，是GIS的核心 |
| 模型 | 空间分析模型、应用模型 |
| 人员 | 系统开发人员、数据生产者、最终用户 |

| GIS功能链 | 说明 |
|-----------|------|
| 数据采集与输入 | 将各种来源数据转为计算机可识别形式 |
| 数据编辑与更新 | 数据检查、修改、维护 |
| 数据存储与管理 | 空间数据库的组织与管理 |
| 空间查询与分析 | GIS核心功能，区别于其他系统的根本标志 |
| 数据与图形的交互显示 | 可视化表达 |
| 应用模型与系统开发 | 面向具体应用的模型构建 |

### 1.3 GIS 发展

| 阶段 | 时间 | 标志 |
|------|------|------|
| 萌芽期 | 1960s | 加拿大CGIS（世界第一个GIS） |
| 发展期 | 1970s | GIS软件商业化 |
| 应用期 | 1980s | 工作站GIS |
| 产业化期 | 1990s- | 桌面GIS、WebGIS |
| 中国起步 | 1970-1980 | 中科院遥感所GIS研究室 |

### 1.4 GIS 与其他系统区别

| 对比 | GIS | CAD | MIS |
|------|-----|-----|-----|
| 数据 | 空间+属性 | 图形为主 | 属性为主 |
| 分析 | 空间分析 | 制图设计 | 统计分析 |
| 坐标 | 地理坐标系 | 局部坐标系 | 无 |
| 拓扑 | 必须建立 | 一般不建立 | 无 |

## 第2章 · 空间数据

### 2.1 空间数据基本概念

| 术语 | 释义 |
|------|------|
| 空间数据 | 描述空间实体的位置、形状、大小及分布特征的数据 |
| 空间数据特征 | 空间特征（位置）、属性特征（性质）、时序特征（时间） |
| 属性数据 | 表示空间实体的属性特征，是对地理空间数据的说明 |
| 地理空间 | 地理现象发生和存在的空间范围，通常指地球表层 |
| 空间实体 | 具有空间位置和属性特征的地理对象（点、线、面、体） |

### 2.2 空间数据采集方法

| 方法 | 说明 |
|------|------|
| 野外采集 | GPS测量、全站仪等 |
| 地图数字化 | 手扶跟踪数字化、扫描矢量化 |
| 摄影测量 | 航空摄影、数字摄影测量 |
| 遥感处理 | 卫星遥感影像解译与分类 |

### 2.3 栅格数据编码

| 编码方法 | 特点 |
|----------|------|
| 直接栅格编码 | 最简单最直接，逐行逐列记录像元值 |
| 游程编码（RLE） | 记录属性值+连续个数，适合大片同值区域 |
| 四叉树编码 | 递归四等分，同值区合并，适合层次性数据 |
| 链码（Freeman） | 用方向码记录边界，适合线状要素 |

### 2.4 矢量数据结构

| 要素 | 表示方法 |
|------|----------|
| 点 | 坐标(X,Y) |
| 线 | 坐标序列 |
| 面 | 闭合坐标序列+拓扑关系 |

### 2.5 矢量 vs 栅格对比（必考）

| 方面 | 矢量 | 栅格 |
|------|------|------|
| 数据结构 | 复杂、精确 | 简单、规则 |
| 数据量 | 小 | 大 |
| 拓扑关系 | 易建立 | 难建立 |
| 图形精度 | 高 | 取决于分辨率 |
| 空间分析 | 叠置分析复杂 | 叠置分析简单 |
| 图形显示 | 精美 | 有锯齿 |
| 适宜对象 | 离散对象 | 连续场数据 |

### 2.6 数据模型

| 模型 | 特点 |
|------|------|
| 层次模型 | 树形结构，一对多 |
| 网络模型 | 图结构，多对多 |
| 关系模型 | 二维表，最广泛使用 |
| 对象模型 | 封装+继承+多态 |

### 2.7 Geodatabase 三级结构

要素类（Feature Class）→ 要素数据集（Feature Dataset）→ Geodatabase

### 2.8 空间数据质量

| 质量要素 | 说明 |
|----------|------|
| 位置精度 | 空间位置的准确程度 |
| 属性精度 | 属性值的准确程度 |
| 逻辑一致性 | 数据间逻辑关系的正确性 |
| 完整性 | 数据是否完整无遗漏 |
| 时间精度 | 数据的时间现势性 |
| 数据世系 | 数据来源和处理历史 |

## 第3章 · 空间数据的转换与处理

### 3.1 空间参考系统

| 术语 | 释义 |
|------|------|
| 空间参考系统 | 确定地理目标平面位置和高程的平面坐标系和高程系的统称 |
| 地球椭球体 | 地球表面几何模型四类——自然表面→大地水准面→地球椭球面→数学模型 |
| 参考椭球体 | 与局部地区大地水准面最佳拟合的地球椭球体，需确定定位与定向 |
| 大地水准面 | 假想平均海水面延伸穿过大陆形成的封闭曲面，是高程的基准面 |
| 参心坐标系 | 原点在参考椭球中心的坐标系（北京54、西安80） |
| 地心坐标系 | 原点在地球质心的坐标系（CGCS2000、WGS84） |

### 3.2 四大坐标系统对比（必考）

| 坐标系 | 类型 | 原点 | 椭球体 | 特点 |
|--------|------|------|--------|------|
| 北京54 | 参心 | 参考椭球中心 | 克拉索夫斯基 | 原苏联体系，精度较低 |
| 西安80 | 参心 | 参考椭球中心 | IUGG1975 | 中国自主建立，精度提高 |
| CGCS2000 | 地心 | 地球质心 | CGCS2000椭球 | 中国新一代地心坐标系 |
| WGS84 | 地心 | 地球质心 | WGS84椭球 | GPS使用，全球统一 |

### 3.3 地图投影

| 投影类型 | 变形特征 | 典型投影 | 适用场景 |
|----------|----------|----------|----------|
| 等角投影 | 角度无变形，面积变形大 | 墨卡托、高斯-克吕格 | 航海、地形图 |
| 等面积投影 | 面积无变形，角度变形大 | 阿尔伯斯 | 面积统计 |
| 任意投影 | 均有变形但较均匀 | 兰伯特 | 中小比例尺 |

### 3.4 高斯-克吕格投影（重点）

- 类型：等角横切椭圆柱投影
- 特点：中央经线无变形，离中央经线越远变形越大
- 分带：6°带（1:2.5万及更小）、3°带（1:1万及更大）
- 我国6°分带：13-23带
- 坐标规定：X轴=赤道，Y轴=中央经线，Y值加500km避免负值

### 3.5 坐标转换方法

| 方法 | 参数 | 适用 |
|------|------|------|
| 四参数变换 | 2平移+1旋转+1缩放 | 平面相似变换，小范围 |
| 七参数变换 | 3平移+3旋转+1缩放 | 三维赫尔默特变换，大范围 |
| 仿射变换 | 6参数 | X/Y方向不同缩放+扭曲+旋转+平移 |
| 相似变换 | 4参数 | 等比例缩放+旋转+平移 |

### 3.6 投影转换方式

| 方式 | 说明 |
|------|------|
| 正解变换 | 直接建立两个投影间的解析关系式 |
| 反解变换 | 先反解出地理坐标，再代入新投影公式 |
| 数值变换 | 利用离散点坐标，数值逼近建立关系式 |
| 解析-数值变换 | 新投影已知，原投影未知，先反解再代入 |

### 3.7 空间内插方法

| 类型 | 方法 |
|------|------|
| 分块内插法 | 线性内插、双线性多项式、二元样条函数 |
| 逐点内插法 | 移动拟合法、距离反比加权（IDW）、克里金法 |
| 整体内插法 | N次多项式拟合 |

### 3.8 空间数据处理三方面

1. **数据变换**：坐标系变换，包括几何纠正和投影转换
2. **数据重构**：格式转换、结构转换、类型替换
3. **数据提取**：类型提取、窗口提取、空间内插

## 第4章 · 空间数据的可视化表达

### 4.1 可视化基本概念

| 术语 | 释义 |
|------|------|
| 数据可视化 | 将数字信息转换为人可识别的图形，清晰有效传递信息 |
| 地理信息可视化三层次 | ①地图数据的可视化表示 ②地理信息的可视化表示（分级+专题地图） ③空间分析结果的可视化表示 |
| 符号化 | 按实际情况确定地图符号基本形状，以颜色或形状区分事物性质 |

### 4.2 符号化8种方法（必考）

| 方法 | 说明 | 适用场景 |
|------|------|----------|
| 单一符号 | 所有要素同一符号 | 仅关注位置分布 |
| 分类符号 | 按属性类别不同符号 | 土地利用分类、行政区划 |
| 分级符号 | 按数值大小不同大小符号 | 人口等级、GDP等级 |
| 分级色彩 | 按数值大小不同颜色 | 温度分布、降水分布 |
| 比率符号 | 符号大小与属性值成比例 | 精确表示数量（值过大时不适用） |
| 组合符号 | 多属性组合表示 | 同时表示多个指标 |
| 统计符号 | 饼图/柱状图符号 | 多属性统计对比 |
| 彩色阴影 | 用色彩深浅表示 | 地形渲染、阴影效果 |

### 4.3 矢量符号化

| 要素类型 | 符号类型 |
|----------|----------|
| 点状 | 标记符号（Marker） |
| 线状 | 线符号（Line） |
| 面状 | 填充符号（Fill） |

### 4.4 栅格符号化

| 类型 | 说明 |
|------|------|
| 分类栅格符号 | 不同类别不同颜色 |
| 分级栅格符号 | 不同级别不同色彩深浅 |

### 4.5 专题地图

| 类型 | 说明 |
|------|------|
| 质量底色图 | 表示现象的质量特征 |
| 等值线图 | 用等值线表示连续变化现象 |
| 分级统计图 | 以行政单元为单位分级着色 |
| 定点符号图 | 用点状符号表示离散现象 |
| 动线图 | 用箭头表示运动方向和路径 |

### 4.6 地图制图综合

- 概括：简化细节，保留主要特征
- 合并：相邻同类要素合并
- 选取：根据重要性取舍
| 移位：避免符号重叠而偏移
