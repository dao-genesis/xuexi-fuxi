## 一、数据可视化概述

### 1.1 目的

> 数据可视化是为了**清晰有效地传递信息**，便于研究者与用户对数据进行分析推理
> 数字信息对计算机可识别，但对人眼不可识别，必须转换为**人可识别的图形**才具有实用价值

### 1.2 地理信息可视化三层次（简答高频）

| 层次 | 内容 |
|------|------|
| **地图数据的可视化** | 地图数据的视觉表达 |
| **地理信息的可视化** | 地理时空数据分级处理 → 选择视觉变量 → 专题地图表示 |
| **空间分析结果的可视化** | 网络分析/缓冲区分析/叠加分析等结果 → 专题地图描述 |

## 二、数据符号化方法（必考核心）

### 2.1 符号化原则

> 按实际情况确定地图符号的基本**形状**，以符号的**颜色或形状**区分事物的性质
> 用点、线、面符号表示呈点、线、面分布特征的地理要素

### 2.2 符号化8种方法（必考·简答高频）

| 方法 | 说明 | 适用场景 |
|------|------|----------|
| **单一符号** | 大小、形状、颜色统一；只反映地理位置，不反映定量差异 | 仅需显示位置 |
| **分类符号** | 按属性类别不同符号；属性值相同→相同符号，不同→不同符号 | 土地利用分类、行政区划 |
| **分级符号** | 按数值大小不同大小符号；符号大小取决于分级数值 | 人口等级、GDP等级 |
| **分级色彩** | 按数值大小不同颜色；配色方案+分级方法决定表达效果 | 温度分布、降水分布 |
| **比率符号** | 属性值与符号大小一一对应；反映同级别间微小差异 | 精确数值表达（值过大时不适用） |
| **组合符号** | 多属性组合表示 | 同时表示多个指标 |
| **统计符号** | 表示多项属性；饼图/柱状图/累计柱状图 | 多属性统计对比 |
| **彩色阴影** | 用色彩深浅表示 | 地形渲染、阴影效果 |

![符号化方法](../006-空间数据的可视化表达-空间数据的可视化表达/page_006.jpg)

> ⚡ 分级色彩 vs 分级符号：色彩用**颜色**区分级别，符号用**大小**区分级别
> ⚡ 比率符号注意：属性值过大时符号过大影响视觉效果，不宜使用

### 2.3 统计符号详解

| 类型 | 用途 |
|------|------|
| **饼图** | 表示整体属性与组成部分之间的**比例关系** |
| **柱状图** | 表示两项可比较的属性或**变化趋势** |
| **累计柱状图** | 既表示相互关系与比例，也表示相互比较与趋势 |

## 三、矢量数据符号化

### 3.1 矢量符号类型

| 要素类型 | 符号类型 |
|----------|----------|
| **点状** | 标记符号（Marker） |
| **线状** | 线符号（Line） |
| **面状** | 填充符号（Fill） |

### 3.2 符号系统

> 菜单 → 自定义(Customize) → 样式管理器(Style Manager)
> 可选择已有符号，或设计新符号加入样本库
> 自定义样式：样式引用 → 创建新样式(***.style) → 新建符号

### 3.3 矢量符号化设置路径

> 图层属性(Layer Properties) → 符号化(Symbology)

| 显示选项 | 对应方法 |
|----------|----------|
| Features → 单一符号 | Single Symbol |
| Categories → 分类色彩/唯一值 | Unique Values |
| Quantities → 分级色彩/分级符号/比率符号 | Graduated Colors / Graduated Symbols / Proportional |
| Charts → 统计符号 | Pie/Bar/Stacked |

## 四、栅格数据符号化

### 4.1 栅格符号化方法

| 方法 | 说明 | 适用数据 |
|------|------|----------|
| **分类栅格符号** | 将栅格值分为若干类，每类一种颜色 | 离散数据（土地利用、行政区划） |
| **分级栅格符号** | 将栅格值按范围分级，每级一种颜色 | 连续数据分级（AQI指数、浓度分布） |
| **拉伸(Stretched)** | 连续变化数据，沿色带拉伸显示 | 高精度连续数据（DEM高程） |

### 4.2 特殊方法 — 透明度

> 图层属性 → 显示(Display) → Transparency设置
> 常用于将栅格图层叠加在矢量图层之上时，设置透明度以同时显示两层信息

## 五、专题地图（简答高频）

### 5.1 专题地图概念

> 突出表示一种或几种主题要素的地图，如人口分布图、土地利用图、等温线图等

### 5.2 专题地图5种类型

| 类型 | 说明 |
|------|------|
| **质量底色图** | 表示现象的质量特征 |
| **等值线图** | 用等值线表示连续变化现象 |
| **分级统计图** | 以行政单元为单位分级着色 |
| **定点符号图** | 用点状符号表示离散现象 |
| **动线图** | 用箭头表示运动方向和路径 |

### 5.3 地图编制过程

> 地图数据符号化与注记标注 → 为地图编制准备基础数据
> 还需完成：**布局纸张设置、制图范围定义、制图比例尺确定、图名/图例/坐标网/指北针**

## 六、地图制图综合（名词解释高频）

> 根据地图用途和比例尺，对地图内容进行选取、概括、合并和移位，以保持地图清晰可读的过程

| 操作 | 说明 |
|------|------|
| **概括** | 简化细节，保留主要特征 |
| **合并** | 相邻同类要素合并 |
| **选取** | 根据重要性取舍 |
| **移位** | 避免符号重叠而偏移 |

## 八、地图装饰要素

### 8.1 地图要素清单

| 要素 | 插入方式 | 说明 |
|------|----------|------|
| **标题(Title)** | Insert → Title | 可设置字体、大小、角度 |
| **指北针(North Arrow)** | Insert → North Arrow | 指示方向，可选样式 |
| **比例尺(Scale Bar)** | Insert → Scale Bar | 可选择样式、单位、主/次刻度 |
| **图例(Legend)** | Insert → Legend | 向导式创建，可选图层、列数、间距 |
| **标签(Label)** | 图层属性 → Labels | 自动标注要素属性 |
| **图片(Picture)** | Insert → Picture | 插入Logo等图片 |
| **对象(Object)** | Insert → Object | 插入Word/Excel等OLE对象 |

### 8.2 图例设置要点

> - 图例向导(Legend Wizard)：选择数据框 → 选择图层 → 设置图例标题字体 → 设置图例框架 → 设置间距
> - 图例属性(Legend Properties)：可调整各部分间距、图例项排列、补丁形状
> - 自定义图例补丁：自定义 → 自定义模式 → 命令 → 页面布局 → 新建图例补丁形状
> - 双击图例 → Properties → 可修改各项详细设置

### 8.3 标签设置要点

> - 图层属性 → Labels → 勾选"Label features in this layer"
> - 标签方法：Label all features the same way
> - 标签字段：选择要显示的属性字段

**标签放置属性**（必考）：

| 要素类型 | 放置选项 |
|----------|----------|
| **点要素** | 优先右上，可设置偏移和角度；优先级0=阻塞，1=最高，3=最低 |
| **线要素** | 水平/平行/曲线/垂直，上方/下方/线上 |
| **面要素** | 水平/直线，仅在多边形内；可先尝试水平再尝试直线 |

## 十、地图导出

### 10.1 导出操作

> File → Export Map
> 可选格式：**PNG、JPEG、PDF、BMP、GIF、TIFF、SVG**
> 关键参数：**分辨率(Resolution)**，默认96 DPI

### 10.2 导出注意事项

| 格式 | 特点 | 适用场景 |
|------|------|----------|
| **PDF** | 支持矢量输出 | 印刷 |
| **PNG/JPEG** | 位图格式 | 屏幕显示、网页 |
| **TIFF** | 无损压缩 | 高质量输出 |
| **SVG** | 矢量格式 | 网页交互 |

> 分辨率越高，文件越大，输出越清晰

## 十一、制图设计原则

### 11.1 制图前四问（简答高频）

1. **目的(Objective)**：到底是为了什么？
2. **受众(Audience)**：受众是谁？期望获取什么信息？是否有特殊需求（如色盲）？
3. **介质(Medium)**：采用什么介质？纸质/屏幕/大尺度打印？
4. **条件(Condition)**：在什么条件下被查看？

### 11.2 数据图层选择

> - 地图是否讲述了一个**吸引人的故事**？
> - 区分**主要图层**和**辅助图层**
> - 一些层次是否会**遮蔽关键信息的可视化**？

### 11.3 数据简化方法

| 方法 | 说明 |
|------|------|
| **查询功能(Query)** | 显示要素的子集（如只显示>6.0级地震） |
| **属性选择(Select by Attributes)** | SQL条件筛选 |
| **导出数据(Export Data)** | 将选中的要素导出为新数据集 |
| **裁剪(Clip to Shape)** | 数据框属性中裁剪到指定形状 |

## 十二、经典试题精选

### 名词解释

1. **数据可视化**：将数字信息转换为人可识别的图形图像，清晰有效地传递信息的过程。
2. **符号化**：按实际情况确定地图符号的基本形状，以颜色或形状区分事物性质，将空间数据转换为地图表达的过程。
3. **专题地图**：突出表示一种或几种主题要素的地图，如人口分布图、土地利用图、等温线图等。
4. **分级色彩**：按数值大小用不同颜色表示的符号化方法，将要素属性按分级方法分成若干级别，不同颜色表示不同级别。
5. **分级符号**：按数值大小用不同大小符号表示的符号化方法，符号大小取决于分级数值的大小或级别高低。
6. **比率符号**：符号大小与属性值成比例的符号化方法，属性值与符号大小一一对应，可反映同级别间微小差异。
7. **制图综合**：根据地图用途和比例尺，对地图内容进行选取、概括、合并和移位，以保持地图清晰可读的过程。

### 简答题

**Q1：简述符号化的8种方法（必考）**
> ①单一符号 ②分类符号 ③分级符号 ④分级色彩 ⑤比率符号 ⑥组合符号 ⑦统计符号 ⑧彩色阴影

**Q2：简述地理信息可视化的三层次**
> ①地图数据的可视化表示 ②地理信息的可视化表示（分级+专题地图） ③空间分析结果的可视化表示

**Q3：简述统计符号的3种类型及其用途**
> ①饼图：表示整体属性与组成部分的比例关系 ②柱状图：表示两项可比较的属性或变化趋势 ③累计柱状图：既表示比例关系，也表示比较与趋势

**Q4：简述专题地图的5种类型**
> ①质量底色图 ②等值线图 ③分级统计图 ④定点符号图 ⑤动线图

**Q5：简述制图综合的4种操作**
> ①概括：简化细节，保留主要特征 ②合并：相邻同类要素合并 ③选取：根据重要性取舍 ④移位：避免符号重叠而偏移

**Q6：简述制图前需要考虑的四个问题**
> ①目的(Objective) ②受众(Audience) ③介质(Medium) ④条件(Condition)

**Q7：简述数据简化的4种方法**
> ①查询功能(Query) ②属性选择(Select by Attributes) ③导出数据(Export Data) ④裁剪(Clip to Shape)
