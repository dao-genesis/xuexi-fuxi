# 实验七 · 空间空值分析（空间插值 + 分区统计 + 栅格计算）

> 对应理论：空间分析（栅格）。以三峡库区 2018 年降雨为例，串联**插值 → 分区统计 → 栅格计算**。

## 一、实验目的
通过实例，掌握**空间插值（Kriging）**、**分区统计（Zonal Statistics）**、**栅格计算**等 ArcGIS 操作及其应用意义。

## 二、数据
- `PRE_2018.csv`：三峡库区 2018 年各站点降雨（字段 `lat / long / elev / year_pre`）。
- `TRG_county`：库区区县边界（含字段 `Name`）。

## 三、内容一 · 空间插值（Kriging）
1. CSV → `Display XY Data`（按 long=X、lat=Y）→ 导出为点要素 `pre_2018.shp`。
2. `Spatial Analyst ▸ Interpolation ▸ **Kriging**`：
   - **Input point features** = `PRE_2018`
   - **Z value field** = `year_pre`（待插值的降雨量）
   - **Output surface raster** = `pre_2018.tif`
   - Kriging method = **Ordinary**；Semivariogram model = **Spherical**
   - **Output cell size** = `150`
   - Search radius = Variable（点数 12）
3. **环境设置 ▸ Raster Analysis ▸ Mask（掩膜）= `TRG_county`**：
   - 有掩膜 → 仅在库区范围内插值（边界裁剪）；
   - 无掩膜 → 输出整个矩形范围。

## 四、内容二 · 分区统计（按区县统计 2018 年均降雨）
- **Zonal Statistics as Table（输出表格）**：`Spatial Analyst ▸ Zonal ▸ Zonal Statistics as Table`：
  - Input zone = `TRG_county`；Zone field = **`Name`**；Input value raster = `pre_2018.tif`；
  - Output table = `tgr.dbf`；Statistics type = **MEAN**；勾选 `Ignore NoData`。
  - 结果：一张表，列出各区县降雨年均值。
- **Zonal Statistics（输出栅格）**：参数同上，但 **Output raster** = `pre_2018_z.tif`（无「输出表」项）：
  - 结果：各区县整体填充其均值的专题栅格图。

## 五、考点与要义
- **空间插值**：由离散采样点推求连续表面（降雨、高程、污染浓度等）。Kriging 利用半变异函数刻画空间自相关，分普通（Ordinary）/ 泛（Universal）克里金。
- **掩膜（Mask）**：限定分析 / 输出范围到指定区域，是按行政边界出图的关键。
- **分区统计两工具区别**：`as Table` 输出**属性表**（便于后续统计 / 连接）；普通 `Zonal Statistics` 输出**栅格**（便于专题成图）。
- **Zone field**：定义「区」的字段（如行政区名 `Name`），每个唯一值是一个统计单元。
- **栅格计算（Raster Calculator）**：对栅格做代数 / 条件运算（如 `Con`），是栅格分析的通用引擎。
