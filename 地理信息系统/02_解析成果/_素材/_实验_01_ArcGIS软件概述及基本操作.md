# 实验一 · ArcGIS 软件概述及基本操作

> 对应理论：第 1 章 地理信息系统概论。先「识软件」，为后续全部上机打基础。

## 一、实验目的
认识 ArcGIS Desktop 三大组件（**ArcMap / ArcCatalog / ArcToolbox**），掌握软件界面构成与最基本的数据加载、浏览、图层管理操作。

## 二、实验内容
1. 启动 ArcMap，认识主界面：菜单栏、工具栏、**目录表（TOC / Table Of Contents）**、显示区。
2. 区分**数据视图（Data View）**与**布局视图（Layout View）**。
3. 加载数据（Add Data），浏览地图（放大 / 缩小 / 漫游 / 全图 / 书签）。
4. 图层管理：显隐、调整叠放顺序、符号化（Symbology）、标注（Label）。
5. 在 ArcCatalog 中浏览与连接数据；认识 ArcToolbox 工具箱体系。

## 三、关键操作
- **加载数据**：工具栏 `Add Data` 按钮 或 菜单 `File ▸ Add Data`，定位到数据所在文件夹（首次需在 ArcCatalog 中 `Connect To Folder` 建立文件夹连接）。
- **视图切换**：窗口左下角图标，或菜单 `View ▸ Data View / Layout View`。数据视图用于浏览分析，布局视图用于排版出图。
- **图层符号化**：图层右键 `Properties ▸ Symbology`，设置单一符号 / 分类 / 分级等。
- **书签**：`Bookmarks ▸ Create Bookmark` 记住某视野范围。

## 四、考点与要义
- **三大组件分工**：ArcMap（显示/编辑/分析/出图）、ArcCatalog（数据管理）、ArcToolbox（地理处理工具）。
- **数据视图 vs 布局视图**：前者看数据本身，后者做地图整饰与打印输出。
- **TOC 作用**：管理图层顺序与显隐——上层图层覆盖下层；面要素通常置于线、点之下。
- **坐标系**：数据框（Data Frame）坐标系决定显示投影；首个加载图层常决定数据框坐标系。
