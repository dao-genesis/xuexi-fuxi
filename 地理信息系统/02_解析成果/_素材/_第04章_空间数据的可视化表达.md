》地理信息的可视化表示：利用各种地理时空数

第一节
数据符号化

---

**p006** ![](../006-空间数据的可视化表达-空间数据的可视化表达/page_006.jpg)

符号化的原则是按实际情况确定地图符号的基本
形状，以符号的颜色或者形状区分事物的性质。
例如用点、线、面符号表示呈点、线、面分布特征
的地理要素，点表示建筑或者特定地点，线表示公
路和铁路，面用来表示地区。
》符号化方法可分为以下几类：单一符号、分类符
号、分级符号、分级色彩、比率符号、组合符号、
统计符号、彩色阴影等。

---

**p007** ![](../006-空间数据的可视化表达-空间数据的可视化表达/page_007.jpg)

符号化方法分类
单一符号：采用大小、形状、颜色都统一的点状、
线状或者面状符号来表达制图要素。
忽略了要素在数量、大小、比例等方面的差异；
）只能反映制图要素的地理位置而不能反映要素的
定量差异。
点值符号：使用一定大小的点状符号来表示一定
数量的制图要素。是一种用点的密度来表现要素
空间分布的方法，数值较大的区域点较多，数值
小的地区点较小。

---

**p008** ![](../006-空间数据的可视化表达-空间数据的可视化表达/page_008.jpg)

符号化方法分类
>分类色彩：根据数据层要素属性值来设置地图
符号的方式是分类符号表示方法。
将不同属性值的要素分开，属性值相同的采用
相同的符号，属性值不同的采用不同的符号；
利用不同形状、大小、颜色、图案的符号来表
达不同的要素；
能够反映出地图要素的数量或者质量的差异

---

**p009** ![](../006-空间数据的可视化表达-空间数据的可视化表达/page_009.jpg)

符号化方法分类
分级色彩：将要素属性数值按照一定的分级方
法分成若干级别之后，用不同的颜色来表示不
同级别。
每个级别用来表示数值的一个范围，从而可以
明确反映制图要素的定量差异；
颜色的选择和分级的设置要取决于制图要素的
特征；
只有合理的配色方案和科学的分级方法才能将
地图中要素的宏观分布规律体现得清晰明确

---

**p010** ![](../006-空间数据的可视化表达-空间数据的可视化表达/page_010.jpg)

符号化方法分类
》分级符号：与分级色彩设置有所不同，是采用
不同大小的符号来表示不同级别的要素属性数值。
符号形状取决于制图要素的特征，符号大小取
决于分级数值的大小或者级别高低；
一般用于表示点状或者线状要素，如多用于表
达人口分级图、道路分级图等；
可以直观地表达制图要素的数值差异；
制图要素分级和分级符号表示是关键环节。

---

**p011** ![](../006-空间数据的可视化表达-空间数据的可视化表达/page_011.jpg)

符号化方法分类
比率符号：按照一定的比率关系，来确定与制图
要素属性数值对应的符号大小，一个属性数值就
对应了一个符号大小，这种一一对应的关系使得
符号设置表现得更细致，不仅反应不同级别的差
异，也能反映同级别之间微小的差异。
如果属性数值过大，则不适合采用此种方法，
因为比率符号过大会严重影响地图的整体视觉
效果。

## 复习要点 · 已填充

> 基于 OCR 全文自动提取，人工校核待补。

### 一、核心概念（名词解释）

- [x] **数据可视化**：将数字信息转换为人可识别的图形，清晰有效传递信息，便于分析推理
- [x] **地理信息可视化三层次**：①地图数据的可视化表示 ②地理信息的可视化表示（分级+专题地图） ③空间分析结果的可视化表示
- [x] **符号化**：按实际情况确定地图符号基本形状，以颜色或形状区分事物性质
- [x] **单一符号**：大小、形状、颜色统一的符号，忽略定量差异，只反映地理位置
- [x] **分类色彩/分类符号**：不同属性值用不同符号，反映数量或质量差异
- [x] **分级色彩**：属性值按分级方法分若干级别，不同颜色表示不同级别，反映定量差异
- [x] **分级符号**：不同大小符号表示不同级别属性值，用于点/线要素
- [x] **比率符号**：属性值与符号大小一一对应，反映同级别间微小差异；数值过大时不适用
- [x] **统计符号**：饼图（整体与部分比例）、柱状图（比较/趋势）、累计柱状图（比例+趋势）
- [x] **点值符号**：用点密度表现要素空间分布，数值大区域点多
- [x] **专题地图**：突出表示一种或几种自然/社会经济现象的地图
- [x] **地图布局要素**：图名、图例、比例尺、坐标网、指北针、图框、附图

### 二、关键公式 / 模型

- [x] **符号化方法分类**：单一符号、分类符号、分级符号、分级色彩、比率符号、组合符号、统计符号、彩色阴影
- [x] **矢量符号化**：点状→标记符号；线状→线符号；面状→填充符号

### 五、思考题 / 自测

- [x] 题：分级色彩与分级符号有何区别？
  答：分级色彩用不同颜色表示级别差异，适用于面状要素；分级符号用不同大小表示，适用于点/线要素
- [x] 题：比率符号与分级符号有何区别？
  答：比率符号一一对应（连续变化），分级符号按级别分组（离散变化）
- [x] 题：制作一幅完整专题地图需要哪些要素？
  答：图名、图例、比例尺、坐标网、指北针、图框，可能还有附图和文字说明
