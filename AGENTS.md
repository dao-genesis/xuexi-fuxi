## 八 · 常见之惑（我或问，先答之）

**Q1：`课程_闭环.py` 与 `期末复习_总枢.py` 有何分？**

A：**视角不同，皆可用**：
- `课程_闭环.py X` —— **单课视角**·直对 `<X>/` 课夹运作，输出 `<X>/00_闭环总览.md`。**主公当下偏此**。
- `期末复习_总枢.py -f X` —— **跨课视角**·以 `解析仓库/` 为枢，输出 `解析仓库/_学期总图.md`。**多课同跑或要 LLM 调度时用此**。

实质：闭环是"为单课自成"，总枢是"为全学期统观"。

**Q2：`01_原始PDF` vs `02_原始课件PDF` 之异？**

A：旧名 vs 新名。`课程_闭环.py` 用 `PDF_DIR_CANDIDATES = ['01_原始PDF', '02_原始课件PDF', '01_原始pdf']` 兼容。新建课夹宜用 `01_原始PDF/`（数字小者优）。

**Q3：「复习要点」未填之章如何识？**

A：`课件_道喂.py` 之占位为 `## 复习占位 · 待 LLM 协填` 或类似；已填则替为 `## 复习要点 · LLM 协填`。`课程_闭环.py._已填复习要点(content)` 是定式判别器（同 `期末_全图._已填复习要点`）。

**Q4：调 LLM 之前要查什么？**

A：(1) `.env` 已配 OPENAI_API_KEY；(2) `LLMClient.from_env('vision').config.is_ready()` 返 True；(3) 看 `.llm_cache/` 是否已有缓存（同问不二答）。

**Q5：何时调 `--rescan`？**

A：改了章节识别规则（如 `_yc_common.cn_to_int` 或 `detect_chapter`）后。**速极快**——不动图像，仅刷 `_doc.json` 之 `chapter_num / chapter_raw / all_chapter_nums`。

**Q6：`_doc.json:pages[*].embedded_text` 何时实？**

A：雨课堂 PDF 本身**无文字层**（图像合成 PDF）·`pdf_提文.py` 取不到字。文层之实**赖道喂关**——`章占位填充任务` 之 schema 含 `页文转录`，LLM 协填时按页录字，由 `课件_道喂.py:在线协填` 或 `课件_回填.py:回填一章` 调 `章占位填充任务.apply_page_texts` 写入。**一关二事**：要点入章 md + 页文入 _doc.json。素材 md 重生时·主版图链下展 `<details>` 折叠之原文（人可读可搜·agent 可入提示）。**「无为而无不为」之实**。详 `全链路_道说.md § 七` 文层。

**Q7：旧 task json 之响应可否照常回填？**

A：可。回填关已兼容——若 task json 缺 `doc_json_path_rel`，自动从 `page_paths` 推 `_doc.json` 路径。但旧 task json 之 schema 无「页文转录」字段·LLM 若按旧 schema 返·则 `result.页文转录 = []`·_doc.json 留 null。**欲实页文须重生 task**：`python 课件_道喂.py --课 X --章 N`（覆盖式重生·不丢历史响应）。

**Q8：`解析仓库/` 与 `<课>/02_解析成果/` 之主从？双轨为何不归一？**

A：**有容乃大·非芜也**。两径所容之物大同小异（289 文件级·24 MB 量级·哈希对照几全等），然其用各异：

- **`<课>/02_解析成果/`（主·课夹自治）**：单课闭环之根·`课程_闭环.py X` 直对此径作八关增量·`<课>/00_闭环总览.md` 自此而生·主公当下日常皆此径。
- **`解析仓库/`（从·跨课总枢）**：跨课总图之枢·`期末复习_总枢.py -f X` 与 `期末_全图.py` 用此径出 `_学期总图.md`·LLM 调度多课时仍以此为枢。

脚本兼容两径于核心常量：`课程_闭环.py:PARSE_DIR_CANDIDATES = ['02_解析成果', '03_解析成果']` 与 `期末_模拟卷.py:_PARSE_DIR_CANDIDATES`（双源扫描·课夹优先·解析仓库兜底）。

**主公已决**于 `环境法学/README.md § 五·B`：「`解析仓库/` 为冗余源·与 `02_解析成果/` 几全等·不需迁移」——`.gitignore` 已收 `解析仓库/`·不入版本控制。**强求归一 = 损脚本之容 = 反「大方无隅」之教**。后人见此双径勿误为芜·**此乃主公早设之有容**。同义之于「`01_原始PDF` vs `02_原始课件PDF`」（见 Q2）：`PDF_DIR_CANDIDATES` 三候选·新建课夹宜用 `01_原始PDF/`·旧课夹保其历史名亦无碍。

> *大方无隅·大器免成。* —— 容两序而不强一，是谓「无为而无不为」。

## 九 · 道之教（终）

> *为学者日益，闻道者日损。损之又损，以至于无为，无为而无不为。*

- **慎终若始**——动手前必先察，损物前必先验
- **执一以为天下牧**——以课夹为单元，以闭环为常态

> *知人者，知也。自知者，明也。*
>
> *道恒无为，而无不为。*
