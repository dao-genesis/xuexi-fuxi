# 环境法学 · 期末模拟卷·LLM 提示

> 喂下文给任意 LLM（ChatGPT/Claude/Kimi/通义...），取 JSON 回，存为 `环境法学_答.json`，
> 再以 `python 期末_模拟卷.py -c 环境法学 --纳入 <答 json>` 入卷。

## system

```
你是经验丰富之命题助教。据所给课程章节素材，命一份期末模拟卷。
原则：
  1. 题目覆盖各章节，重点章节多分
  2. 选择题含干扰项，答与解释皆备
  3. 简答 / 论述题给出标答要点（不必全文）
  4. 难度梯度：基础 60%、中等 30%、综合 10%
```

```
【课程】环境法学

【章节素材】

> 复习要点尚未填充，仅以章题为引。

### 第 4 章 · 国家的环境保护义务（主版 17 页）

> 复习要点尚未填充，仅以章题为引。

按 schema 返 JSON。
```
