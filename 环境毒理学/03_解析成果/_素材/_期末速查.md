# 环境毒理学 · 期末速查表

> 一页之表，含各章关键概念、公式、考点。已吸纳 7/7 章复习要点。

| 章 | 标题 | 核心概念（≤3） | 关键公式 / 模型 | 高频考点 |
| ---: | ---- | -------------- | ---------------- | -------- |
| 1 | 绪论 | 环境毒理学（environmental toxicology）：研究环境物理性、化学性和生物污染物，特别是环境化学性污染物对人类健康的毒性作用及其机理的科学。也称"环境健康毒理学"。；环境污染物：由于人为或自然原因进入环境并使环境的正常组成和性质发生改变、直接或间接有害于人类与其他生物的物质。；一次污染物（原生污染物）：由污染源直接排入环境，其物理和化学性状未发生变化的污染物。 | Paracelsus 剂量定律："所有物质都是毒物，没有物质不是毒物，唯一的区别是它们的剂量"——抛开剂量谈毒性都是耍流氓。这是毒理学最根本的原理。；环境污染分类模型：按污染物性质→化学污染、物理污染、生物污染；按污染物形态→废气、废水、固体废弃物、噪声、辐射污染。；环境毒理学研究方法体系：体内试验（in vivo）+ 体外试验（in vitro）+ 流行病学调查，三个层次互补。 | 环境毒理学定义三要素：环境污染物（特别是化学污染物）→ 人类健康 → 毒性作用及机理；环境化学污染物是种类最多、污染最严重、分布最广、危害最严重的，是环境毒理学主要研究对象，简称环境化学物；污染源四类：工业、农业、交通运输、生活 |
| 2 | 污染物在环境中的迁移和转化 | 污染物迁移：污染物在环境中发生的空间位置相对改变的过程；污染物转化：污染物在环境中通过物理、化学或生物学作用改变形态或转变为另一种物质的过程；机械性迁移：通过气（扩散+气流搬运）、水、重力等物理作用使污染物空间位移 | BCF（生物富集系数） = 生物体内浓度(mg/kg) / 环境中浓度(mg/kg)；BAF（生物积累系数） = 发育后阶段体内浓度 / 发育前阶段体内浓度；BMF（生物放大系数） = 较高营养级浓度 / 较低营养级浓度 | 迁移三类型：机械性（气/水/重力）、物理-化学性（6种作用）、生物性（浓缩/积累/放大）；气的机械性迁移三扩散：风力、气流、湍流；物理-化学性迁移六作用：风化淋溶、溶解挥发、酸碱、络合、吸附、氧化-还原 |
| 3 | 污染物的生物转运与生物转化 | 生物转运：环境污染物经各种途径和方式同生物体接触而被吸收、分布和排泄等过程的总称（反复通过生物膜）；生物转化/代谢转化：化学物在组织细胞中发生的结构和性质的变化过程；被动扩散（简单扩散）：物质从高浓度→低浓度透过生物膜，纯物理过程，不耗能 | 一级动力学消除：dD/dt = -KD，D = D₀·e^(-kt)，lgD = lgD₀ - kt/2.303；半衰期：T₁/₂ = 0.693/K；表观分布容积：Vd = D/c（Vd大→分布广/组织吸收多/血浆浓度低；Vd小→分布少/血浆浓度高） | 生物转运四方式：被动扩散（不耗能，顺浓度）、滤过（水溶性）、特殊转运（主动+易化）、膜动转运（胞吞/胞吐）；被动扩散影响因素：浓度梯度、脂/水分配系数、电离状态与pH、蛋白结合亲和力；主动转运五特点：逆浓度梯度、耗能、需载体、有选择性、有容量限制+竞争性抑制 |
| 4 | 环境污染物的毒作用及其影响因素 | 急性毒性：机体一次接触或24h内多次接触某一化学物所引起的毒性效应（包括死亡）；LD₅₀/LC₅₀：半数致死量/半数致死浓度，使半数试验动物死亡的毒物剂量/浓度，数值越小毒性越强；亚慢性毒性：机体在相当于1/20左右生命期间，少量、反复接触有害因素引起的损害作用 | LD₅₀：半数致死量（mg/kg），数值越小毒性越强；蓄积系数：K = ∑LD₅₀(n) / LD₅₀(1)；K<1 高度蓄积  /  1~3 明显蓄积  /  3~5 中度蓄积  /  >5 轻度蓄积 | 一般毒性vs特殊毒性：一般=急性/亚慢性/慢性/蓄积；特殊=致畸/致癌/致突变/生殖/发育；急性毒性试验：测定LD₅₀，观察期一般14天，选成年健康小鼠雌雄各半；亚慢性毒性：染毒1-3个月，选年幼初断奶动物，2剂量组+1对照组 |
| 5 | 环境毒理学常用实验方法 | 动物染毒途径：经口（灌胃）、经皮肤、经呼吸道吸入；实验动物采血法（小鼠/大鼠尾静脉、豚鼠耳缘剪口、家兔耳静脉）；LD₅₀ 测定与急性毒性试验设计 | LD₅₀ 测定法：概率单位法 / 改良寇氏法（Karber）/ 霍恩法（Horn）/ 上下法；灌胃参考量：小鼠 0.1 mL/10 g 体重（最大 ≤ 0.5 mL/10 g） | LD₅₀ 测定经典法：概率单位法、改良寇氏法、霍恩法、上下法；急性毒性观察期 = 14 d；毒性分级：极毒 < 1 mg/kg < 剧毒 < 50 < 中等 < 500 < 低 < 5000 < 微 |
| 6 | 环境化学物的安全性和健康危险度评价 | 安全性评价（Safety Evaluation）：依据毒理学试验结果，结合人群暴露资料，评定化学物在一定接触水平下对人体健康是否安全之过程；危险度评价（Risk Assessment）：定量估测某接触水平下某化学物致害人群健康之概率与程度之过程；危险度（Risk）：在特定条件下接触某物，发生有害效应之概率（单位时间、人群、效应） | UF：不确定系数（10×10×10×10 ≤ 10000，常见 100~1000）；MF：修正因子（1~10）；危险商 HQ = ADD / RfD：HQ ≤ 1 安全；> 1 风险升 | 安全性评价四阶段法：急性 → 遗传+亚急 → 亚慢+生殖 → 慢性+致癌；危险度评价 NAS 四步法：危害识别 → 剂量反应 → 暴露评价 → 危险表征；RfD = NOAEL / UF（必背） |
| 7 | 常见化学致癌物的环境毒理学 | 化学致癌作用（Chemical Carcinogenesis）：化学物诱发正常细胞转化为肿瘤细胞之过程；致癌物（Carcinogen）：能引起或诱发动物或人类肿瘤之化学物；直接致癌物：本身即活性形式，不需代谢活化（如烷化剂、亚硝胺活性中间体） | —（本章以致癌机制与分类为主；致癌风险 R = SF × ADD 见第六章危险度评价） | 化学致癌三阶段：启动 → 促进 → 进展；直接致癌物不需代谢活化；间接致癌物需 P450 活化；苯并[a]芘 → BPDE → 与 DNA 鸟嘌呤共价结合 |


## 名词速对（≤30）

| 术语 | 释义 | 章 |
| ---- | ---- | ---: |
| 环境毒理学（environmental toxicology） | 研究环境物理性、化学性和生物污染物，特别是环境化学性污染物对人类健康的毒性作用及其机理的科学。也称"环境健康毒理学"。 | 1 |
| 环境污染物 | 由于人为或自然原因进入环境并使环境的正常组成和性质发生改变、直接或间接有害于人类与其他生物的物质。 | 1 |
| 一次污染物（原生污染物） | 由污染源直接排入环境，其物理和化学性状未发生变化的污染物。 | 1 |
| 污染物迁移 | 污染物在环境中发生的空间位置相对改变的过程 | 2 |
| 污染物转化 | 污染物在环境中通过物理、化学或生物学作用改变形态或转变为另一种物质的过程 | 2 |
| 机械性迁移 | 通过气（扩散+气流搬运）、水、重力等物理作用使污染物空间位移 | 2 |

| 危险度（Risk） | 在特定条件下接触某物，发生有害效应之概率（单位时间、人群、效应） | 6 |
| 化学致癌作用（Chemical Carcinogenesis） | 化学物诱发正常细胞转化为肿瘤细胞之过程 | 7 |
| 致癌物（Carcinogen） | 能引起或诱发动物或人类肿瘤之化学物 | 7 |
