## 二、必拿分的"四条主线"（先背这些）

1. **剂量决定毒性**：Paracelsus 定律 → 剂量–效应关系 vs 剂量–反应关系（S形曲线）→ LD₅₀、阈剂量、NOAEL/LOAEL → ADI = NOAEL / SF。
2. **毒物在体内的旅程（ADME）**：吸收 → 分布 → 代谢（生物转化 I/II 相）→ 排泄；跨膜转运四方式、四大贮存库、肝肠循环。
3. **毒作用与机制**：选择性毒性、靶器官、生物标志物；影响因素（化学结构、剂量、联合作用：相加/协同/拮抗/独立）。
4. **安全与风险**：健康危险度评价四步（危害鉴定 → 剂量–反应评定 → 暴露评定 → 危险度表征）；致癌物分类与三致（致癌/致畸/致突变）。

## 四、必会计算（考前再算一遍）

- **ADI**：`ADI = NOAEL / SF`（SF 常取 100）；每日允许总量 = ADI × 体重。
- **一级消除动力学**：`C = C₀·e^(−kt)`；`lgC = lgC₀ − kt/2.303`（直线斜率 = **−k/2.303**，不是 −k）；半衰期 `T₁/₂ = 0.693/k`；表观分布容积 `Vd = D/C`。

## 五、易错雷区（临考扫一眼）

- lgC–t 直线**斜率是 −k/2.303**，常被写成 −k → 必错。
- 剂量–**效应**关系（个体效应强度，计量资料）≠ 剂量–**反应**关系（群体出现比例，计数资料）。
- **易化扩散**顺浓度、不耗能；**主动转运**逆浓度、耗能、可饱和可竞争——别混。
- 肠胃是吸收/排泄**通道**，不是贮存库；四大贮存库 = 血浆蛋白 / 肝肾 / 脂肪 / 骨骼。
- 颗粒物 **< 2.5 μm 沉积肺泡**；消化道吸收主要在**小肠**。
