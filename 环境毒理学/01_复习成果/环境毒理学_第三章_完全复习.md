## 一、核心概念

### 生物转运 vs 生物转化

| 概念 | 定义 | 本质 |
|------|------|------|
| **生物转运** | 污染物经各种途径同生物体接触而被**吸收、分布和排泄**的总称 | 反复通过生物膜（位置变化） |
| **生物转化** | 化学物在组织细胞中发生的**结构和性质的变化**，又称代谢转化 | 化学结构变化 |

### 机体处置毒物四步

**吸收 → 分布 → 代谢 → 排泄**

环境化学物对机体毒性作用的大小和部位，与其在体内的吸收、分布、代谢和排泄过程有密切关系。

## 二、跨膜转运机理 (3.1.1)

### 生物膜结构
细胞膜主要由**脂质双分子层**和**蛋白质**组成，也有少量糖形成糖脂和糖蛋白。

### 四种转运方式对比

| 特征 | 被动扩散 | 滤过 | 主动转运 | 易化扩散 | 膜动转运 |
|------|--------|------|--------|--------|--------|
| 浓度梯度 | 顺 | 顺压差 | **逆** | 顺 | — |
| 能量消耗 | 否 | 否 | **是** | 否 | 是 |
| 载体 | 否 | 否 | 是 | 是 | 否 |
| 选择性 | 低 | 分子大小 | 高 | 高 | — |
| 竞争抑制 | 否 | 否 | 是 | 是 | — |

### ① 被动扩散 (passive diffusion)
- 物质只能从浓度**高→低**透过生物膜
- **纯物理过程，不消耗代谢能，膜不具有主动性**
- 影响因素：浓度梯度差 / 脂/水分配系数 / 离解状态和pH / 蛋白结合亲和力

### ② 滤过 (infiltration)
- 水溶性毒物随水分子经生物膜**孔状结构**透过
- 动力：流体静压梯度差 + 渗透压梯度差

### ③ 特殊转运 (载体转运)
适用于脂溶性低或分子量过大的化学物。

**主动转运** (active transport)：逆浓度梯度，消耗能量，需载体，有选择性，有容量限制，可竞争性抑制（5大特点）

**易化扩散** (facilitated diffusion)：载体介导但**顺浓度，不消耗能量**，不能逆浓度梯度

### ④ 膜动转运
- 颗粒物和大分子物质，利用膜的可塑性和流动性
- **胞吞作用**（外→内）和 **胞吐作用**（内→外）

## 三、吸收 (3.1.2)

**定义：** 污染物通过接触部位透过体内生物膜**进入血液循环**的过程。

### 3.1 呼吸道吸收
- 气体/蒸汽→肺泡→**被动扩散**→血液
- 颗粒物按粒径沉积：>10μm阻挡鼻外 / 2.5-10μm上呼吸道 / **<2.5μm附着肺泡**
- 颗粒归宿：①进入血液 ②进入消化道 ③进入淋巴腺 ④长期滞留引起病变
- 影响因素：分压差和血/气分配系数 / 溶解度和分子量 / 肺泡通气量与血液流量比值
- 案例：**尘肺**

### 3.2 消化道吸收
- 主要吸收部位是**小肠**，胃里主要通过**简单扩散**
- 影响因素：酶类和菌丛 / 内容物种类数量 / 溶解度和分散度 / 酸碱度（分子态>电离态）
- 案例：**清华学生铊中毒**

### 3.3 皮肤吸收
- 途径①：毒物→角质层→透明层→颗粒层→生发层→基膜→真皮
- 途径②：通过汗腺、皮脂腺、毛囊绕过表皮进入真皮
- 两阶段：**穿透相**（速度与脂溶性正比）→ **吸收相**（真皮→毛细血管）
- 影响因素：皮肤完整度 / 部位（背部>手臂）/ 化合物物化性质 / 接触条件
- 案例：**儿童铅中毒**

## 四、分布 (3.1.3)

- 起始分布取决于**血流量**，最终分布取决于与组织器官的**亲和力**

### 屏障
- **血脑屏障** / **胎盘屏障**

### 四大贮存库

| 贮存库 | 要点 |
|--------|------|
| **血浆蛋白** | 与蛋白结合的毒物不易透过细胞膜→保护；可逆非共价键结合→可解离再分布。例：DDE竞争置换胆红素→黄疸 |
| **肝肾** | 含特殊结合蛋白，能夺取血浆蛋白结合的毒物；配体蛋白结合有机酸；金属硫蛋白结合Zn/Cd起解毒作用 |
| **脂肪组织** | 脂溶性毒物贮存（DDT/六六六/有机汞）。降低靶器官浓度，但脂肪消耗时毒物释放→毒性显现 |
| **骨骼组织** | 氟化物/铅/锶与骨基质结合。氟→氟骨症，锶→骨肉瘤 |

### 贮存的毒理学意义（双刃剑）
- **保护**：急性中毒时降低体液毒物浓度，减少到达靶器官的量
- **危害**：慢性毒物来源。例：铅贮存于骨骼→缺钙/pH下降时释放→慢性中毒

> **练习答案：** 毒物贮藏库→**ABD**（骨骼、脂肪、肝肾），肠胃不是贮存库

## 五、排泄 (3.1.4)

**定义：** 毒物母体化学物及其代谢产物向机体外转运的过程，是代谢的**最后一个环节**。

### 5.1 经肾排出
- **肾小球滤过**：可滤过分子量<69000的物质，与血浆蛋白结合的不能通过
- **肾小管重吸收**：极性亲水→不被重吸收→随尿排出；非极性亲脂→重吸收回血液
- **肾小管主动分泌**：有机阴/阳离子主动分泌到尿中

### 5.2 经肝胆排出
- **肝肠循环**：脂溶性毒物→小肠重吸收→门静脉→回到肝脏→再随胆汁分泌→循环。延长体内停留时间，半衰期延长，毒性加大
- **直接排出**：高度极性化合物不易被小肠吸收→随胆汁混入粪便排出

### 5.3 经呼吸道排出
- 主要通过**简单扩散**，速度取决于分压差和血气分配系数
- 颗粒物通过清除系统→咽部→咳出或吞入消化道

### 5.4 其他途径
- 乳腺（有机碱/亲脂性毒物）/ 唾液腺汗腺（I/Br/F/Hg）/ 头发指甲（重金属）
- 排出时局部毒性：镉→损害近曲肾小管 / 汞→口腔炎 / β-萘胺→膀胱癌

## 六、代谢动力学 (3.3)

### 6.1 毒物动力学
运用数学方法，定量研究毒物的吸收、分布、排泄和代谢转化随时间动态变化的规律。

### 6.2 室的概念
- **一室模型**：转运速率高→体内分布迅速达到平衡→视为一室
- **二室模型**：**中央室**（血流丰富，迅速平衡）+ **周边室**（血流少，穿透慢）。大多数毒物复合二室模型

### 6.3 一级动力学核心公式

| 公式 | 含义 |
|------|------|
| dD/dt = −KD | 消除速度与体内毒物量成正比 |
| D = D₀·e^(−kt) | 毒物量随时间指数衰减 |
| lgD = lgD₀ − kt/2.303 | 对数-时间直线 |
| lgc = lgc₀ − kt/2.303 | 浓度对数-时间直线 |

**判断方法：** lgc对t作图得直线→一级动力学消除。**斜率 = −k/2.303**（不是−k！）

### 6.4 四个主要参数

| 参数 | 公式 | 意义 |
|------|------|------|
| **表观分布容积 Vd** | D/c | Vd大→分布广、组织吸收多、血浆浓度低 |
| **生物半衰期 T₁/₂** | 0.693/K | 体内毒物量减少一半所需时间 |
| **消除速率常数 K** | (dD/dt)/D | K越大→消除越快 |
| **消除率 Cl** | (dD/dt)/c | 反映血浆消除毒物的能力 |

### 6.5 非线性动力学
线性模型假设吸收无限、速率与浓度成正比。实际上计量加大→出现**饱和现象**→非线性动力学。

## 七、课堂练习题详解

### 题1（多选·p36）
> 以下哪些是毒物在体内的贮藏库？A.骨骼 B.脂肪 C.肠胃 D.肝肾

**答案：ABD** — 肠胃是吸收排泄通道，不是贮存库。

### 题2（选择·p49）
> k=0.3465 h⁻¹，c₀=100 mg/L，下列说法**错误**的是？

- A. 半衰期为2小时 → T½=0.693/0.3465=2h ✓
- B. 4小时后浓度为25mg/L → c=100×e^(−0.3465×4)=100×0.25=25 ✓
- **C. lgc对t直线斜率为−0.3465** → 实际斜率=−k/2.303=−0.1504 ✗ **答案选C**
- D. 消除速度与当前浓度成正比 → 一级动力学特征 ✓

### 题3（填空·p50）
1. 一级动力学微分方程：**dD/dt = −KD**，指数形式：**D = D₀·e^(−kt)**
2. lgc对t得到一条**直线**→一级动力学；直线方程：**lgc = lgc₀ − kt/2.303**
3. 消除速率常数k与半衰期换算：**T₁/₂ = 0.693/k**

## 八、高频考点速记

1. 被动扩散是毒物跨膜转运**最主要**的方式
2. 主动转运五大特点：逆浓度/消耗能量/需载体/有选择性/竞争性抑制
3. 颗粒物<2.5μm→沉积在**肺泡**
4. 消化道吸收主要部位→**小肠**
5. 皮肤吸收穿透速度与**脂溶性正比**
6. 起始分布取决于**血流量**，最终分布取决于**亲和力**
7. 四大贮存库：血浆蛋白/肝肾/脂肪/骨骼
8. 贮存库双刃剑：急性保护 vs 慢性危害
9. **肝肠循环**延长毒物体内停留时间→半衰期延长→毒性加大
10. lgc-t直线斜率 = **−k/2.303**（不是−k）
11. T₁/₂ = **0.693/K**
12. Vd = **D/c**
