# 雨课堂 PDF 集尽 — 使用说明

> 反者道之动也；弱者道之用也。
> 道法自然，无为而无不为。

## 一、其要

此脚本不与雨课堂之安全验证争。
循"上善若水"之理：
1. 汝在自家浏览器中扫码登录雨课堂（顺人之道）
2. 脚本承汝之登录 cookie 而潜行雨课堂之 API（顺机之道）
3. 自动遍历汝所有课程，下载所有课件并合成 PDF（无为而无不为）

## 二、依赖（一次安之）

```powershell
pip install requests pillow
```

可选（自动从浏览器取 cookies，省下手动复制）：

```powershell
pip install browser-cookie3
```

## 三、用法

### 道一：最简（一劳永逸 · 强推）

```powershell
python 雨课堂_PDF_集尽.py
```

执行后：
- **首次运行**：自动启 Chromium 浏览器，汝扫码登录即可
- **登录后**：脚本自取 sessionid 并存于 `.session_cache.txt`
- **次次运行**：自动复用缓存，**无须再扫码**

### 道二：显式启用 playwright 登录

```powershell
python 雨课堂_PDF_集尽.py --login
```

### 道三：直输 sessionid（若已有）

```powershell
python 雨课堂_PDF_集尽.py --sessionid <你的sessionid>
```

### 道四：禁用自动登录（仅试浏览器/手动）

```powershell
python 雨课堂_PDF_集尽.py --no-auto-login
```

### 道五：过滤课程（仅下某课）

```powershell
python 雨课堂_PDF_集尽.py --filter "环境毒理学"
python 雨课堂_PDF_集尽.py --filter "毒理" --filter "化学"
```

### 道六：交互选择

```powershell
python 雨课堂_PDF_集尽.py --select
```

会列出所有课程，让你选要下哪些（例：`1,3,5-7`）。

### 道七：仅看课程列表（不下载）

```powershell
python 雨课堂_PDF_集尽.py --list-only
```

### 道八：自定输出目录

```powershell
python 雨课堂_PDF_集尽.py --output "E:\我的课件"
```

## 四、手取 sessionid 之法

1. 浏览器中登录: <https://www.yuketang.cn/v2/web/index>
2. 按下 `F12` 打开开发者工具
3. 切至 `Application` (Edge/Chrome) 或 `存储` (Firefox)
4. 左侧 `Cookies` → 选 `https://www.yuketang.cn`
5. 右表中找 `sessionid` 一行
6. 复制 `Cookie Value` 一列之值（约 32 字符长）

## 五、特性

- ✅ **跳过已下**：重复运行无虞，已存在的 PDF 自动跳过
- ✅ **并行下载**：每张幻灯片同时下，疾如雷
- ✅ **v3/v1 兜底**：新旧两种 API 并试，无所不取
- ✅ **多课程批量**：一次取尽你所有课程
- ✅ **归档清晰**：每门课一文件夹，每节课一 PDF
- ✅ **三类登录**：从浏览器自取 / 手输 / 命令行参数

## 七、可能之问

**Q: 出现 `登录态无效`？**
A: sessionid 过期或错。重新登录浏览器，重新取 sessionid。

**Q: 雨课堂提示 `429 Too Many Requests`？**
A: 慢下。可降并发：编辑脚本内 `max_workers=8` 改为 `4`。

**Q: 某些课件未下载？**
A: MOOC 类型课时（type=15, 17）本无 PPT，自动跳过。
   若是 type=2 的"剧本"课件，本脚本暂不支持（需 selenium），将跳过。

**Q: 想下视频？**
A: 此脚本专为 PDF 而设，循"少则得，多则惑"。
   若需视频，参 GitHub: `84634E1A607A/rainclassroom-scrapper`。
