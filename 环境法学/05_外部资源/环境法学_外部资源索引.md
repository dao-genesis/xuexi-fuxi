### E. 智慧树知到 · 单元测试答案（已爬取）

> 来源：[新浪博客](https://blog.sina.com.cn/s/blog_aa02a2100102zmix.html)

**第四章 环境标准与环境监测**：
- 环境基准不具有法律效力 → **对**
- 环境标准实施监督部门：国家环保总局 + 县级以上人民政府环保部门
- 国家环境标准统一编号发布：**国家环保总局**
- 环保标准样品标准没有国家标准只直接针对企业 → **错**
- 按监测目的分类 → **错**（应分为监视性/研究性/特定目的）

**第五章 环境法律责任**：
- 环境民事责任主要是一种财产责任 → **对**
- 即使排污行为未造成损害后果，行为人仍应承担环境民事责任 → **错**
- 环境民事责任适用**无过错责任原则**

**国际环境法**：
- 环境和资源主权原则对发展中国家特别重要 → **对**
- 解决全球环境问题只是发达国家责任 → **错**
- 国际环境法基本原则：尊重国家主权 / 新的全球伙伴精神 / 公平承担责任 / 和平解决争端

**自然资源保护法**：
- 原则：开发利用和保护相结合 / 因地制宜合理开发 / 统筹兼顾综合利用循环利用
- 少开发少利用**不是**自然资源保护法原则
- 自然资源费只能专款专用 → **对**
- 不可再生资源：合理利用 + 节约利用

## 八 · 核心名词解释速查（从网络资料汇编）

> 以下为高频考点名词，从各来源综合整理

| 术语 | 释义 |
|------|------|
| 环境法基本原则 | 调整因开发利用、保护和改善环境而产生的社会关系的指导思想和基本准则 |
| 预防原则 | 预防为主，防治结合，综合治理 |
| 协调发展原则 | 经济、社会发展与环境保护相协调 |
| 原因者负担原则 | 污染者付费、利用者补偿、开发者保护、破坏者恢复 |
| 公众参与原则 | 公民有权参与环境决策、监督和诉讼 |
| 环境影响评价制度 | 对规划和建设项目实施后可能造成的环境影响进行分析、预测和评估，提出对策和措施的制度 |
| "三同时"制度 | 建设项目中防治污染的设施应与主体工程同时设计、同时施工、同时投产使用 |
| 排污许可制度 | 排污者须向环保部门申请并获取排污许可证后方可排放污染物 |
| 限期治理制度 | 对严重污染环境的单位责令限期完成治理任务 |
| 环境标准 | 环境管理目标和效果的表示，包括环境质量标准和污染物排放标准 |
| 环境基准 | 环境中有害物质不对人体和生态产生不良影响的剂量-效应关系（不具有法律效力） |
| 排污权 | 经环保部门许可，以污染物排放控制标准为限向环境排放污染物的权利 |
| 排污交易 | 在保持一定区域内污染物排放总量不变的条件下，排污者间买卖排污权 |
| 环境民事责任 | 无过错责任原则，举证责任倒置 |
| 环境法律关系 | 环境法主体之间因环境利用、保护和改善形成的权利义务关系 |
| 污染环境罪 | 违反国家规定排放有害物质严重污染环境的行为（刑法第338条） |
| 自然保护区 | 对有代表性的自然生态系统等划出予以特殊保护和管理的区域 |
| 生态保护红线 | 在重点生态功能区、生态环境敏感区和脆弱区等区域划定实行严格保护 |
