## 四、一令贯通

```bash
# 全链路·增量（八关：提文→聚合→素材→道喂→回填→汇编→模拟卷→总览）
python 课程_闭环.py 环境法学

# 加：离线道喂任务（导出 .json+grid.jpg 至全局 .道喂任务/）
python 课程_闭环.py 环境法学 --feed offline --grid

# 加：在线 LLM 协填（须 .env 配 API key）
python 课程_闭环.py 环境法学 --feed online --grid

# 加：在线 LLM 出期末模拟卷（写至 _素材/_期末模拟卷.md）
python 课程_闭环.py 环境法学 --mock online

# 仅重扫章节信号（速极快）
python 课程_闭环.py 环境法学 --rescan

# 单独跑期末模拟卷（与课程_闭环 之第八关 等效；双源扫描）
python 期末_模拟卷.py -c 环境法学                  # 离线
python 期末_模拟卷.py -c 环境法学 --在线            # 在线
python 期末_模拟卷.py --列                          # 列所有可生卷之课
```
