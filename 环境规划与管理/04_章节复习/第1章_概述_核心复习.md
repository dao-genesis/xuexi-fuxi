## 核心知识

### 一、环境管理

> **🅽 名词解释标准答案**：运用技术、经济、法律、行政、教育等手段，对人类损害环境质量的活动施加影响，协调发展与环境的关系，达到既要发展经济满足人类的基本需要，又不超过环境的容许极限。

**理解五要点**：
1. 协调发展与环境的关系
2. 运用各种手段限制人类损害环境质量的行为
3. 跨学科领域的新兴综合学科
4. 动态过程
5. 需要各国协调合作

> **内涵**：规范和限制人类的观念和行为
> **核心**：对**人类损害环境质量的活动**进行管理

⚠️ **易错**：环境管理≠治理污染，而是转变观念和调整行为（判断题陷阱）

---

### 二、环境规划

> **🅽 名词解释标准答案**：国民经济和社会发展规划的有机组成部分，是指为使环境与社会经济协调发展，把"社会—经济—环境"作为一个复合生态系统，依据社会经济规律、生态规律和地学原理，对其发展变化趋势进行研究而对人类自身活动和环境所作出的时间和空间上的合理安排。

**五层含义**：
1. 国民经济和社会发展规划的**有机组成部分**
2. 环境管理的**首要职能**
3. 环境决策在**时间、空间**上的具体安排
4. 带有**指令性**的环境保护方案
5. **实质**：克服人类经济社会活动和环境保护活动**盲目性和主观随意性**的**科学决策活动**

⚠️ **易错**：环境规划的实质=**科学决策活动**（非经济发展规划/污染治理方案/环境监测计划）

---

### 三、环境规划与环境管理的关系⭐⭐⭐

| 关系 | 说明 |
|------|------|
| 规划职能是环境管理的**首要职能** | 规划是核心内容和基础，管理以规划为依据 |
| 环境目标是两者的**共同核心** | 都围绕实现环境质量目标展开 |
| 具有**共同的理论基础** | 生态学、系统论、可持续发展等 |

### 四、五要素与五手段

**五要素**：人（第一主要对象）、物（重要研究对象）、资金（物质基础）、信息（"神经"）、时空（突出特点）

**五手段**：

| 手段 | 核心 |
|------|------|
| 行政 | 组织和指挥权力 |
| 法律 | **强制性** |
| 经济 | 价值规律，价格/税收/信贷 |
| 技术 | 提高生产率+控制污染 |
| 宣传教育 | 普及知识+思想动员 |

⚠️ **易错**：环境管理基本手段**不包括军事手段**（选择题陷阱）

---

### 五、环境规划与管理的任务与目的

**基本任务**：转变人类社会的基本观念和调整人类社会的行为
**长期根本任务**：环境文化建设（文化决定行为）

**三种社会行为**：政府行为、市场行为、公众行为

**根本目的**：创建可持续的发展模式和消费模式，符合人与自然和谐相处的原则

**四大作用**：
1. 促进环境与经济、社会可持续发展
2. 保障环保活动纳入国民经济和社会发展计划
3. 实施环境政策、法规和制度的主要途径
4. 实现以较小投资获取较佳环境效益

---

### 六、环境容量与环境承载力⭐⭐⭐

**环境容量**：
- 定义：某环境单元给定功能区目标和质量目标下所允许承纳的污染物质的**最大数量**
- 公式：**M = K + R**（稀释容量 + 自净容量）
- 属性：自然生态环境的基本属性之一
- 特征：地带性规律、地区性差异

> **🅽 名词解释**：在人类生存和自然生态系统不致受害的前提下，某一环境单元所能容纳的污染物的最大负荷量。

**水环境容量**：稀释容量K（稀释作用）+ 自净容量R（沉降、生化、吸附等）

**环境承载力**：
- 定义：自然环境能够承载和支撑的人类社会活动的**强度和总量**
- 公式：**EBC = f(T, S, B)**（时间、空间、人类经济行为规模与方向）
- 是判断人类活动是否对自然环境构成威胁的**基本标准**

> **🅽 名词解释**：在某一时期、某种状态或条件下，某地区的环境所能承受的人类活动作用的阈值。

**承载力指标体系（三部分）**：

| 指标类型 | 示例 |
|----------|------|
| 资源供给指标 | 水资源、土地资源、生物资源数量质量及开发程度 |
| 社会影响指标 | 经济总量、污染治理投资、公用设施水平、人口密度 |
| 污染容纳指标 | 污染物排放量、绿化状况、污染物净化能力 |

**承载力特征**：客观性和主观性、区域性和时间性、动态性和可调控性

**容量 vs 承载力**：

| | 环境容量 | 环境承载力 |
|---|----------|------------|
| 侧重 | **自然属性**（内在禀赋） | **社会属性**（外在禀赋） |
| 性质 | 纯自然量 | 非纯自然也非纯社会 |
| 有限性 | 相对确定、有限 | 有限但可通过人类行为动态调控 |

⚠️ **易错**：环境承载力≠环境容量（承载力是阈值概念，容量是负荷量概念，两者相关但有区别）

### 七、生态学理论

**三定律（米勒）**：

| 定律 | 内容 | 别称 |
|------|------|------|
| 一 | 任何行动都不是孤立的，对自然界的任何侵犯具有无数效应，许多不可逆 | 多效应/极限性原理 |
| 二 | 每种事物无不与其他事物相互联系和交融 | 相互联系/生态链原理 |
| 三 | 生产的任何物质均不应对生物地球化学循环有干扰 | 勿干扰/生物多样性原理 |

**极限性原理 → 环境容量与环境承载力**（见上节）

**生态链原理 → 工业生态学与生态工业园**：
- 工业生态学核心："从摇篮到坟墓"的全过程管理系统观
- 主要方法：工业代谢分析、生命周期评价（LCA）
- 典范：丹麦**卡伦堡**生态工业园（世界最早最著名）
- 我国首个：**广西贵港**国家生态工业（制糖）示范园区（2001年）

**生物多样性原理 → 环境伦理观**：

| 伦理观 | 核心主张 | 代表 |
|--------|----------|------|
| 生命中心主义 | 所有生命都应受到尊重 | 保罗·泰勒《尊重自然》 |
| 地球整体主义 | "大地伦理"，人从征服者→共同体成员 | 李奥波德 |
| 代际均等伦理观 | 当代人与后代人享有同等权利 | — |

### 八、可持续发展理论

> **🅽 名词解释**：既能满足当代人的需要，又不对后代人满足其需要的能力构成危害的发展（1987年《我们共同的未来》）

**四原则**：协调、公平、持续、共同

**标志**：资源的**永续利用**和良好的生态环境

**基本思想**：
1. 不否定经济增长，但重新审视如何实现
2. 以提高生活质量为目标
3. 承认自然环境的价值
4. 以自然资源为基础，同环境承载能力相协调

⚠️ **易错**：可持续发展的核心理念=**资源的可持续利用**（非经济增长/社会公平/环境保护）

---

### 九、复合生态系统论

**复合生态系统 = 自然子系统 + 经济子系统 + 社会子系统**

四大功能：生产、生活、还原、信息传递

**内部矛盾**（需要进行环境规划与管理的根本原因）：
1. 要求自然环境相对稳定而实际剧变
2. 自然环境改变的快速性和恢复调节的缓慢性
3. 资源有限而人类需求无限
4. 地球体积、物质有限而人口增长无限

---

### 十、"两山论"

**三个发展阶段**：
1. 既要绿水青山，也要金山银山
2. 宁要绿水青山，不要金山银山
3. 绿水青山就是金山银山

**本质**：人与自然和谐共处
**核心思想**：经济发展与生态环境保护互动双赢
**科学内涵**：生动诠释了生态经济的自然资本观

---

### 十一、环境经济学原理

**核心**：环境问题实质上是一个**经济问题**

**五大理论**：

| 理论 | 核心 |
|------|------|
| 经济效率理论 | 帕累托最适度 |
| 外部性理论 | 正外部性（环境改善）+负外部性（污染破坏） |
| 物质平衡理论 | 废弃物物质形态不消失，100%回收不可能 |
| 自然环境价值理论 | 使用价值+选择价值+存在价值 |
| 排污交易权理论 | 政府拥有环境资源所有权，市场促外部性内部化 |

---

### 十二、管理二重性原理

| 属性 | 内容 |
|------|------|
| 自然属性 | 同生产力、社会化大生产相联系（技术性） |
| 社会属性 | 同生产关系、社会制度相联系（意识形态性） |

> 两者相互联系、相互制约，像一辆车的两个轮子

**三大职能**：计划（首要性）→ 组织 → 控制

**反馈控制 vs 前馈控制**：

| | 反馈控制 | 前馈控制 |
|---|----------|----------|
| 方式 | 用过去指导现在和将来 | 利用最新信息预测，指导将来 |
| 应用 | 环境质量标准 | 环境规划、环境影响评价 |
| 特点 | 存在时滞 | 解决时滞问题 |

## 📝 本章简答/论述题

### Q1: 环境规划与环境管理的关系（⭐⭐⭐⭐⭐）

1. 规划职能是环境管理的**首要职能**——规划是核心内容和基础，管理以规划为依据
2. 环境目标是两者的**共同核心**——都围绕实现环境质量目标展开
3. 两者具有**共同的理论基础**——生态学、系统论、可持续发展等

### Q2: 环境容量与环境承载力的关系（⭐⭐⭐⭐）

- 容量侧重**自然属性**（内在禀赋），承载力侧重**社会属性**（外在禀赋）
- 容量是纯自然量，承载力非纯自然也非纯社会
- 两者都有限，但承载力可通过人类行为**动态调控**
- 承载力以容量为基础

### Q3: 环境规划与管理的任务（⭐⭐⭐）

- 基本任务：转变人类社会的基本观念和调整人类社会的行为
- 长期根本任务：环境文化建设
- 三种社会行为：政府行为、市场行为、公众行为

### Q4: 管理二重性及三大职能（⭐⭐⭐）

- 自然属性（技术性）+社会属性（意识形态性），像一辆车的两个轮子
- 计划职能（首要性、目标性、普遍性）→组织职能→控制职能
- 控制类型：预防性/更正性、预先/过程/事后、反馈/前馈

### Q5: 论述"公有物悲剧"及环境问题产生原因（⭐⭐⭐⭐）

**"公有物悲剧"**：资源为公共所有时，每个使用者都有动机过度使用（个人获利、集体承担成本），导致资源枯竭。

**环境问题原因**：
1. **市场失灵**：环境资源缺乏明确产权，价格机制失灵
2. **政府失灵**：政策不完善、执行不到位、地方保护主义
3. **人类观念偏差**：传统发展观忽视生态规律
4. **技术两面性**：既可解决也可产生新问题
5. **人口增长与消费升级**

**规划与管理的应对**：科学决策→完善制度→总量控制→功能区划→公众参与→可持续发展
