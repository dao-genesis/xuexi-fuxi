## 核心知识

### 第一节 环境现状调查与评价

#### 一、环境现状调查

> **🅽 名词解释标准答案**：环境现状调查是为了掌握和了解某区域环境现状，发现和识别主要环境问题，确定主要污染源和主要污染物，为环境规划与管理的制定和实施创造条件。

**三方法**⭐⭐⭐⭐⭐：

| 方法 | 优点 | 缺点 | 适用 |
|------|------|------|------|
| **收集资料法** | 省时省力省钱 | 仅获二手资料，不全面 | 初步调查首选 |
| **现场调查法** | 第一手资料，全面 | 工作量大，费时费钱 | 重点区域详细调查 |
| **遥感法** | 远距离综合探测，快速 | 精度受限，需地面验证 | 大范围宏观调查 |

**六内容**⭐⭐⭐⭐⭐（口诀：**特生污质效管**）：

1. **环境特征调查**：自然环境（地理/地质/气候/水文/土壤/生物）+ 社会环境（人口/产业/城镇/交通）
2. **生态调查**：物种/种群/群落/生态系统多样性
3. **污染源调查**：工业/农业/生活/交通污染源
4. **环境质量调查**：大气/水/土壤/噪声等要素质量
5. **环保措施效果调查**：治理项目效果评估
6. **环境管理现状调查**：管理机构/制度/人员/经费

---

#### 二、环境评价

> **🅽 名词解释标准答案**：环境评价是对被评价对象的优劣、好坏作定量或定性描述，一般以定量为主。含自然环境评价、社会经济评价、环境质量评价和污染源评价4方面。

**四方面评价**：

| 评价类型 | 主要方法 |
|----------|----------|
| **自然环境评价** | 类比分析、列表清单、生态图法、指数法与综合指数法、景观生态学法、AHP、生物生产力法、其他（回归/聚类等） |
| **社会经济评价** | 人口/经济/基础设施/土地利用评价 |
| **环境质量评价** | 环境质量指数法 Iᵢ=Cᵢ/C₀ᵢ |
| **污染源评价** | 排毒指数法、环境污染能力潜在指数法、**等标污染负荷法** |

**自然环境评价八法**⭐⭐⭐（选择/填空高频）：

1. 类比分析法
2. 列表清单法
3. 生态图法（指标法+重叠法）
4. 指数法与综合指数法
5. 景观生态学法
6. 层次分析法（AHP）
7. 生物生产力法
8. 其他（回归分析、聚类分析等）

#### 三、等标污染负荷⭐⭐⭐⭐⭐

> **🅽 名词解释标准答案**：等标污染负荷是将不同种类、不同量纲的污染物排放量标准化处理的方法，使各种污染物可以相互比较和加总。

**核心公式**：

| 公式 | 表达 | 说明 |
|------|------|------|
| **环境质量指数** | Iᵢ = Cᵢ / C₀ᵢ | Cᵢ=监测值，C₀ᵢ=标准值 |
| **等标污染负荷（废水）** | Pᵢ = (Cᵢ/C₀ᵢ) × Qᵢ | Qᵢ=废水排放量(m³/d) |
| **等标污染负荷（废气）** | Pᵢ = (Cᵢ/C₀ᵢ) × qᵢ | qᵢ=废气排放流量(m³/d) |
| **污染负荷比** | Kⱼ = Pⱼ/Pₜ × 100% | 累计≥80%→主要污染源 |
| **污染源等标污染负荷** | Pₙ = ΣPᵢ | 某污染源所有污染物等标负荷之和 |
| **区域等标污染负荷** | Pₜ = ΣPₙ | 区域所有污染源等标负荷之和 |

⚠️ **易错**：等标污染负荷法是**污染源评价**方法，不是环境质量评价方法

**主要污染源判定**：按污染负荷比从大到小排序，累计≥80%的污染源即为主要污染源

**污染源评价三法**⭐⭐⭐：
1. 排毒指数法
2. 环境污染能力潜在指数法
3. 等标污染负荷法（最常用）

### 第二节 环境统计方法

> **🅽 名词解释标准答案**：环境统计是用数字反映并计量人类活动引起的环境变化，是对环境系统的信息采集、处理和分析的系统性方法。

#### 一、环境统计指标体系七子系统⭐⭐⭐⭐

| # | 子系统 | 核心内容 |
|---|--------|----------|
| 1 | 工业污染防治 | 工业废水/废气/固废排放量与处理率 |
| 2 | 生活污染防治 | 城镇生活污水/垃圾产生与处理 |
| 3 | 农业污染防治 | 化肥/农药使用量，畜禽养殖污染 |
| 4 | 治理投资 | 环保投资总额及占GDP比例 |
| 5 | 自然生态保护 | 自然保护区面积/数量，生态修复面积 |
| 6 | 环境管理 | 环评审批、排污许可、执法处罚 |
| 7 | 自身建设 | 环保机构人员、装备、经费 |

#### 二、环境统计三研究法⭐⭐⭐

1. **大量观察法**：对大量环境现象进行观察，消除偶然因素影响
2. **综合分析法**：对统计数据进行综合加工和系统分析
3. **归纳推断法**：由样本推断总体

#### 三、环境统计调查三方法

| 方法 | 说明 |
|------|------|
| 统计报表 | 定期上报，制度性调查 |
| 普查 | 专门组织的全面调查 |
| 抽样调查 | 随机抽样推断总体 |

#### 四、环境统计工作程序

统计任务制定 → 统计设计 → 统计调查 → 统计整理 → 统计分析 → 统计资料提供

### 第三节 社会经济预测方法

#### 一、人口预测⭐⭐⭐⭐

| 方法 | 公式 | 说明 |
|------|------|------|
| **算术级数法** | Nₜ = N₀ + b(t-t₀) | b=年均增长人数，线性增长 |
| **几何级数法** | Nₜ = N₀(1+K)^(t-t₀) | K=年均增长率，指数增长 |
| **指数法** | Nₜ = N₀·e^[K(t-t₀)] | 连续复利模型 |

⚠️ **易错**：算术级数=线性增长，几何级数=离散指数增长，指数法=连续指数增长——三者递进关系

#### 二、经济预测

| 方法 | 公式/说明 |
|------|-----------|
| **弹性系数法** | β = e·α（e=能源消费弹性系数0.4~1.1，α=GDP增长率） |
| **能源消费预测** | Eₜ = E₀(1+β)^(t-t₀) |
| **民用耗煤量** | Eₛ = Aₛ·S（Aₛ=人均耗煤量，S=人口数） |

### 第四节 环境预测方法

> **🅽 名词解释标准答案**：环境预测是借助数学、计算理论和信息处理技术，对未来环境质量变化趋势进行定性和定量分析。**预测是决策的基础**。

#### 一、环境预测分类⭐⭐⭐⭐

| 分类方式 | 类型 |
|----------|------|
| **按内容** | 区域环境预测、行业环境预测、专题环境预测 |
| **按作用** | 定性预测、定量预测、定时预测、定比预测、评价预测 |
| **按时间** | 短期（<5年）、中期（5~15年）、长期（≥20年） |

#### 二、定性预测四法⭐⭐⭐⭐

| 方法 | 核心 | 特点 |
|------|------|------|
| **专家会议法** | 专家面对面讨论 | 简便但易受权威影响 |
| **德尔菲法** | 无记名数轮函询 | 不具名+反馈性+统计性 |
| **主观概率法** | 专家主观估计概率 | 定量化主观判断 |
| **层次分析法(AHP)** | 递阶层次+判断矩阵 | 1~9标度，求特征向量得权重 |

> **🅽 德尔菲法标准答案**：又称"专家调查法"，是以无记名方式通过数轮函询征求专家意见的方法。三大特点：①不具名（消除权威影响）；②反馈性（多轮反馈趋同）；③统计性（对意见进行统计处理）。

> **🅽 AHP标准答案**：层次分析法是用递阶层次结构和矩阵方程将思维过程数学化的方法。步骤：建立层次结构→构造判断矩阵(1~9标度)→求特征向量→得权重→一致性检验。

#### 三、定量预测三法⭐⭐⭐⭐

| 方法 | 说明 |
|------|------|
| **时间序列法** | 利用历史数据外推未来（移动平均/指数平滑） |
| **逻辑型规律法（S规律）** | 增长曲线呈S形，适用于环境承载力和技术发展预测 |
| **数学模型法** | 回归分析、系统动力学模型等 |

#### 四、环境质量预测

| 预测类型 | 方法 |
|----------|------|
| 大气质量预测 | 高斯扩散模型、箱模型 |
| 水质量预测 | 数学模型（Streeter-Phelps等）、因果关系分析 |
| 固体废物预测 | 产生量预测+大气/水影响预测模型 |
| 噪声预测 | 多元回归预测、灰色预测方法 |

#### 五、环境预测工作程序⭐⭐⭐

```
确定预测目的 → 确定预测时间 → 制定预测计划 → 收集预测资料
→ 资料分析检验 → 选择预测方法 → 建立预测模型 → 预测分析
→ 预测结果检验 → 输出预测结果 → 提交预测结果
```

### 第五节 决策分析方法⭐⭐⭐⭐⭐

> **🅽 名词解释标准答案**：决策是人们为了达到一定目标，从若干可能的策略中选取最好策略的过程。决策分析是研究怎样才能作出科学决策的系统分析方法。

**决策特点**：多目标性、长期性、不确定性、方案多样性、一次性

#### 一、决策分类⭐⭐⭐⭐⭐

| 类型 | 特征 | 说明 |
|------|------|------|
| **确定型** | 自然状态完全确定 | 结果唯一确定 |
| **风险型** | 状态概率已知 | 可计算期望值 |
| **不确定型** | 状态概率未知 | 靠主观判断 |

⚠️ **易错**：风险型vs不确定型的**根本区别**在于——各自然状态发生的概率**是否已知**

#### 二、决策三要素⭐⭐⭐⭐⭐

| 要素 | 说明 | 可控性 |
|------|------|--------|
| **自然状态** | 决策者无法控制的外部条件 | 不可控 |
| **策略** | 决策者可采取的行动方案 | 可控 |
| **益损值** | 不同策略在不同自然状态下的收益/损失 | — |

#### 三、确定型决策

四个特征：①只有一个确定的自然状态；②存在明确目标；③存在≥2个行动方案且结果确定；④各方案收益/损失可计算。

方法：单纯择优法、模型决策法、排队决策法、盈亏平衡分析

#### 四、风险型决策⭐⭐⭐⭐

| 准则 | 公式 | 说明 |
|------|------|------|
| **最大收益期望值** | Aₙ = max(ΣPᵢaᵢ) | 选期望收益最大的方案 |
| **最小机会损失期望值** | Aₙ = min(ΣPᵢOᵢ) | 选期望损失最小的方案 |

> 两种准则选择结果**通常一致**（互为对偶）

#### 五、不确定型决策五准则⭐⭐⭐⭐⭐

| 准则 | 别名 | 操作 | 口诀 |
|------|------|------|------|
| **等可能性** | Laplace | 每种状态概率=1/n，算期望 | 一视同仁 |
| **乐观准则** | Hurwicz(maximax) | 各方案取最大值，再取最大 | **大中取大** |
| **悲观准则** | Wald(minimax) | 各方案取最小值，再取最大 | **小中取大** |
| **折衷准则** | Hurwicz折衷 | Cᵢ=α·max+(1-α)·min | 乐观悲观折衷 |
| **后悔值准则** | Savage | 各状态最大-各值→后悔矩阵→行max→列min | **最小最大后悔值** |

**折衷公式**：Cᵢ = α·max{aᵢ} + (1-α)·min{aᵢ}（0≤α≤1）
- α=1 → 乐观准则
- α=0 → 悲观准则

**后悔值计算步骤**：
1. 每列（状态）找最大效益值
2. 后悔值 = 列最大值 - 各方案值
3. 每行（方案）找最大后悔值
4. 选各行最大后悔值中最小者

⚠️ **易错**：不同准则可能选出不同方案——为了更准确可靠，应**设法了解概率**，将不确定转化为风险型

#### 六、决策六原则⭐⭐⭐

1. 信息快准全原则
2. 预测未来原则
3. 可行性原则
4. 系统性原则
5. 对比优选原则
6. 集团决策原则

#### 七、决策七程序⭐⭐⭐

| 步骤 | 内容 |
|------|------|
| 1 | 明确问题 |
| 2 | 确定目标（注意先进性与可靠性） |
| 3 | 制定方案（建立模型，提出多种可行方案） |
| 4 | 方案评估（科学计算，数量分析比较优劣） |
| 5 | 选择方案（综合系统分析，选取最优） |
| 6 | 组织实施（落实执行单位，明确责任） |
| 7 | 反馈调整（收集执行信息，分析修正） |

### 第六节 环境审计方法

> **🅽 名词解释标准答案**：环境审计是对特定项目的环境保护情况，包括组织机构、管理、生产及环保设施运转与排污等情况进行系统的、有文字记录的、定期的、客观的评定。

**四个关键词**：系统的、有文字记录的、定期的、客观的

**环境审计也是管理工具**：用于对环境组织、环境管理和仪器设备是否发挥作用进行系统的、定期的和客观的评价。

#### 一、环境审计分类

| 分类方式 | 类型 |
|----------|------|
| 按范围 | 地区（城市）级、工厂/工艺/特定污染物级 |
| 按目标 | 提高管理效率、有效控制污染、提高资金效率、减少事故 |
| 按目的 | 审查执法、废物减量化、实施清洁生产 |

#### 二、环境审计七内容⭐⭐⭐⭐

| # | 审计内容 | 核心 |
|---|----------|------|
| 1 | **符合性审计** | 环保法规符合性评价 |
| 2 | **环境保护管理系统审计** | 管理系统是否运作良好 |
| 3 | **过渡审计** | 不动产获取/剥离的环境风险 |
| 4 | **有害物质处理存放清理审计** | 危险原料全环节审计 |
| 5 | **污染预防审计** | 确定减少废物机会 |
| 6 | **环境效益审计** | 环保措施成本合理性与治理效益 |
| 7 | **产品审计** | 产品与环境政策符合性 |

#### 三、环境审计方法⭐⭐⭐

**常规审计方法**（同样适用环境审计）：
- 审计检查法（资料检查法、实物检查法）
- 审计调查法（查询法、观察法、专题调查法）
- 审计分析法（账户分析法、账龄分析法）

- 经济活动分析法
- 经济技术分析法

### 四、名词解释

1. **环境现状调查**：为掌握和了解某区域环境现状，发现和识别主要环境问题，确定主要污染源和主要污染物，为环境规划与管理的制定和实施创造条件。

2. **等标污染负荷**：将不同种类、不同量纲的污染物排放量标准化处理的方法，Pᵢ=(Cᵢ/C₀ᵢ)×Qᵢ，使各种污染物可相互比较和加总。

3. **环境预测**：借助数学、计算理论和信息处理技术，对未来环境质量变化趋势进行定性和定量分析。预测是决策的基础。

4. **德尔菲法**：又称专家调查法，以无记名方式通过数轮函询征求专家意见。特点：不具名、反馈性、统计性。

5. **决策**：人们为了达到一定目标，从若干可能策略中选取最好策略的过程。特点：多目标性、长期性、不确定性、方案多样性、一次性。

6. **环境审计**：对特定项目环保情况进行系统的、有文字记录的、定期的、客观的评定。

7. **EMIS**：环境管理信息系统，以现代数据库技术为核心，实现环境信息输入/输出/修改/删除/传输/检索/计算等操作的技术工程系统，既是数据库又是管理政策的实验室。

8. **EDSS**：环境决策支持系统，将决策支持系统引入环境规划管理，对结构化/未结构化问题进行描述组织，协助完成管理决策的支持技术。

9. **清洁生产审计**：对产品生产全过程重点环节污染定量监测，找出高物耗、高能耗、高污染原因，提出对策减少污染物产生，实现环境效益与经济效益和谐统一。

10. **层次分析法(AHP)**：用递阶层次结构和矩阵方程将思维过程数学化的方法，1~9标度构造判断矩阵，求特征向量得权重。

### 五、简答题

#### 1. 简述环境现状调查的内容和方法。

**方法**（3种）：
1. 收集资料法——省时省力，仅二手资料，初步调查首选
2. 现场调查法——第一手资料，工作量大，重点区域详细调查
3. 遥感法——远距离综合探测，精度受限需验证，大范围宏观调查

**内容**（6项）：
1. 环境特征调查（自然+社会环境）
2. 生态调查
3. 污染源调查
4. 环境质量调查
5. 环保措施效果调查
6. 环境管理现状调查

#### 2. 简述环境统计的内容及研究方法。

**内容**（指标体系7个子系统）：工业污染防治、生活污染防治、农业污染防治、治理投资、自然生态保护、环境管理、自身建设

**研究方法**（3种）：
1. 大量观察法——对大量环境现象观察，消除偶然因素
2. 综合分析法——对统计数据进行综合加工和系统分析
3. 归纳推断法——由样本推断总体

#### 3. 简述不确定型决策的五种准则。

1. **等可能性准则**：认为各自然状态概率相等(1/n)，计算各方案期望值选最大
2. **乐观准则（大中取大）**：各方案取最大效益值，再从中取最大
3. **悲观准则（小中取大/Wald准则）**：各方案取最小效益值，再从中取最大
4. **折衷准则**：Cᵢ=α·max+(1-α)·min，α为乐观系数(0≤α≤1)
5. **后悔值准则（Savage准则）**：计算后悔矩阵，选各方案最大后悔值中最小者

#### 4. 简述环境审计的内容和方法。

**内容**（7项）：符合性审计、环保管理系统审计、过渡审计、有害物质处理存放清理审计、污染预防审计、环境效益审计、产品审计

**方法**：
- 常规方法：审计检查法、审计调查法、审计分析法、逻辑推理分析法、经济活动分析法、经济技术分析法、数学分析法、抽样审计法
- 三种结合方式：①微观审计与宏观分析相结合；②财务收支审计与经济效益审计相结合；③审计人员与环保专业技术人员相结合

#### 5. 简述环境管理信息系统的组成和设计方法。

**组成**：工作人员、设备（计算机、网络技术、GIS技术、模型库等）及环境原始信息

**设计四阶段**：
1. 可行性研究——提出请求→可行性研究→批准
2. 系统分析——调查→明确目标→信息流分析→逻辑设计
3. 系统设计——初步设计→详细设计→程序设计
4. 系统实施与评价——测试→运行维护→评价验收

**设计原则**：实用性、标准性、先进性、动态性、开放性、经济性、安全性
