## 第一节 大气环境规划

### 一、主要内容与类型

**主要内容6步**：查清问题→确定目标→建立质量模型→制定控制方案→方案优选→实施

**规划类型2类**：

| 类型 | 定义 | 特点 |
|------|------|------|
| **大气环境质量规划** | 以城市总体规划和空气质量标准为依据，规定不同功能区浓度限值 | 管理基础，建立排放-质量关系 |
| **大气污染控制规划** | 实现大气环境质量的方案 | 新建城市→合理布局；已污染城市→经济可行方案 |

### 二、污染源调查与评价

**污染物分类**：气溶胶状(PM10/降尘/TSP/PM2.5) + 气体状态(SO₂/NOx/COx/碳氢/卤族)

**污染源分类**：按来源=固定源(工业+生活)/流动源(交通)；按形状=点/线/面/体源；按高度=高/中/低架源；按时间=连续/瞬时源；按形式=有组织/无组织

**调查内容**：①画污染源分布图(标明位置/排放方式) ②点源(坐标/高度/内径/温度/速度/排放量) ③面源(网格化统计排放量/高度/分类)

**评价方法**：①等标污染负荷法(排序确定主要污染源和污染物) ②污染物排放量排序(最简单方法)

### 三、大气污染预测

**源强预测公式**：Qᵢ = Kᵢ·W·(1-ηᵢ)

- Qᵢ=源强，Kᵢ=排放因子，W=燃料消耗量，ηᵢ=净化去除效率(%)

**质量预测模型**：箱式模型 + 扩散模型

### 四、规划目标与指标体系

**目标2类**：①大气环境质量目标(基本目标，依地域功能区不同) ②总量控制目标(以环境容量为基础，超容量部分为削减量)

**指标体系5类**：

| # | 类型 | 内容 |
|---|------|------|
| 1 | 气象气候 | 气温/气压/风向/风速/风频/日照/大气稳定度/混合层高度 |
| 2 | 大气环境质量 | TSP/飘尘/SO₂/降尘/NOx/CO/光化学氧化剂/O₃/氟化物/苯并[a]芘/细菌总数 |
| 3 | 大气污染控制 | 废气总量/SO₂排放及回收率/烟尘粉尘排放回收去除率/烟尘控制区覆盖率/工业汽车尾气达标率 |
| 4 | 城市环境建设 | 气化率/集中供热率/型煤普及率/绿地覆盖率/人均公共绿地 |
| 5 | 城市社会经济 | GNP/人均GNP/工业产值/行业产值能耗/万元产值能耗/人口密度分布 |

### 五、总量控制方法（3种）

| 方法 | 原理 |
|------|------|
| **A值法** | 地区系数法，根据控制区面积和总量控制系数A计算总允许排放量(宏观箱模式) |
| **P值法** | 根据烟囱有效高度估算各源允许排放量(用烟囱高度控制排放率) |
| **A-P值法** | A值法算总量+修正P值法分配到每个源(二者结合) |
| **平权分配法** | 基于多源模式，按各源对控制点浓度贡献率削减 |
| **优化方法** | 环境效益+经济费用结合，治理费用最小方案 |

### 六、防治措施（4条）

1. **全面规划，合理布局** — 综合运用各种措施组合优化
2. **以集中控制为主** — 宏观调控，节约资源，便于污染防治
3. **强化污染源治理** — 烟尘/SO₂/NOx治理技术
4. **综合防治汽车尾气** — ①立法管理 ②提高油品/清洁燃料 ③发展公交绿色出行

## 第二节 水环境规划

### 一、规划基础

**规划内容6步**：明确问题(水量/水质/利用)→明确目标→选定方法(最优化/模拟优化)→拟定措施→方案比选→实施

**水功能区划**：按主导功能划定并执行相应质量标准的特定区域

**区划原则5条**：①优先保护饮用水源地 ②不降低现有水质等级 ③上下游兼顾 ④与工业布局结合 ⑤实用可行

**水功能分区**：

| 级别 | 类型 |
|------|------|
| 一级区(4类) | 保护区、缓冲区、开发利用区、保留区 |
| 二级区(7类) | 饮用水源区、工业用水区、农业用水区、渔业用水区、景观娱乐用水区、过渡区、排污控制区 |

**水污染控制单元**：可操作实体=排放源+受纳水体；在行政区划基础上结合水体功能划分

**水环境容量**：保证特定水质目标前提下接受的最大污染物量

- 水环境容量=完全分散排放对应的容纳量(极限值)
- 最大允许纳污量=给定排放方式下允许排放的量
- **容量是允许纳污量的极限值**

**计算步骤**：功能区划定目标→划分控制单元简化水体→调查水文和污染源→选水质模型计算

**允许排放量分配3策略**：

| 策略 | 思路 | 举例 |
|------|------|------|
| 公平 | 各源间公平分配 | 等比例削减/按产值或影响确定排污量 |
| 效率 | 削减效率最高 | 区域总费用最小/削减总量最小/总效益最大 |
| 混合 | 公平+效率 | 区间效率策略+区内公平策略(等比例削减) |

### 二、污染源调查与评价

- 调查分类：工业水污染源+生活水污染源
- 方法：收集资料法+现场实测法
- 评价：多采用等标污染指数评价法

### 三、水污染物预测

| 预测项 | 公式 | 说明 |
|--------|------|------|
| 工业用水量 | Q<sub>工</sub>=α·G<sub>t</sub> | α=万元产值用水系数, G<sub>t</sub>=工业产值 |
| 生活用水量 | Q<sub>生</sub>=365·A·D<sub>t</sub> | A=人口(万人), D<sub>t</sub>=人均用水量 |
| 工业废水 | W<sub>ti</sub>=Q<sub>ti</sub>·c<sub>t</sub> | c<sub>t</sub>=排水系数 |
| 工业污染物 | P<sub>ti</sub>=β<sub>t</sub>·G<sub>t</sub> | β<sub>t</sub>=万元产值排放量 |
| 生活污染物 | P<sub>tu</sub>=3.65·A·E<sub>tu</sub> | E<sub>tu</sub>=人均污染负荷 |

### 四、水污染控制规划方法

**最优化法3类**：

| 类型 | 别名 | 要点 |
|------|------|------|
| 排污口最优化处理 | 水质规划 | 求解各排放口处理效率最佳组合，费用最低，规模不变 |
| 最优化均匀处理 | 厂群规划 | 同一效率下寻求最佳位置+规模+管线组合，全区域费用最低 |
| 区域最优化处理 | 区域最优 | 前二者综合，既考虑位置数量又考虑效率 |

**模拟优选法**：定性+定量结合，解一般非最优，取决于规划人员经验

### 五、综合防治措施

**减少排放**：①节约用水(提高重复率/水权制度) ②清洁生产(调整结构/无毒替代/生命周期) ③加强污水处理(有毒物单独处理/集中处理) ④控制面源(农村/农业/畜禽，生态农业示范)

**利用自净**：①人工复氧(增氧器提高DO) ②污水调节(枯水蓄存/丰水释放) ③流量调控(提高枯水期流量)

## 第三节 固体废物管理规划

### 一、概述

**定义**：一定时间和地点无法利用而丢弃的固体/半固体物质。概念具**时间性和空间性**——"放在错误地点的原料"

**分类3类**：

| 类别 | 特点 |
|------|------|
| 城市生活垃圾 | 成分复杂，有机物含量高，元素以碳为主 |
| 工业固废 | 冶金/能源/石化/矿业/轻工等，不同行业种类性质迥异 |
| 危险废物 | 6特性：急性毒性/易燃性/反应性/腐蚀性/浸出毒性/疾病传染性；巴塞尔公约45+2类废物，14种危险特性 |

**危害**：大气(颗粒/毒气/恶臭)→水(地表水/地下水)→土壤(占田/污染)→土地资源→人体健康

### 二、现状调查

| 类型 | 内容 |
|------|------|
| 生活垃圾 | 城市概况/产生特征/收集方式/环卫体系/无害处理/回收体系/处理率利用率 |
| 工业固废 | 有害物质种类/固废种类/产生量/利用量/流向/贮存/处理处置 |
| 危险废物 | 种类产生量/收集运输/综合利用/贮存/处理处置/安全处置率 |

### 三、产生量预测

**生活垃圾**：W = G·K₀·(1+α)^Δt (G=人口, K₀=基准人均量, α=增长量, Δt=年差)

**工业固废(产值法)**：DW<sub>t</sub>=W<sub>t</sub>·S<sub>t</sub>，W<sub>t</sub>=W₀·e^λΔt，S<sub>t</sub>=S₀·e^(-kΔt)

**危险废物**：Q<sub>危</sub>=A·W<sub>t</sub>，W<sub>t</sub>=W₀·e^λΔt

### 四、规划目标与指标

**原则5条**：①源头控制优先促清洁生产 ②因地制宜因废制宜 ③资源化利用产业化 ④全过程控制防转嫁 ⑤集中+点源结合

**指标3类9项**：生活垃圾(无害化率/资源化率/分类收集率) + 工业固废(减量率/综合利用率/处置利用率) + 危险废物(处置利用率/医疗垃圾处理率)

### 五、防治对策

| 类型 | 对策 |
|------|------|
| 生活垃圾 | ①源头控制产生量 ②统筹建设收集运输处置设施+综合利用 ③提出处置方案 |
| 工业固废 | ①制定执行技术政策 ②源头控制减少产生量/调整结构 ③提高利用率 ④加强安全处置 |
| 危险废物 | ①确定名称种类 ②减量化资源化 ③收集运输贮存 ④焚烧+安全填埋 |

### 六、选址方法（5条件）

| 条件 | 关键数字 |
|------|----------|
| 与区域距离 | 下风向，人畜居栖**500m**外 |
| 交通运输 | 距公路/铁路/河流≤**500m** |
| 环境保护 | 使用**15-20年**；距地表水>**100m**；距机场>**10km** |
| 场地建设 | 坡度≤**5%** |
| 地质环境 | K≤**10⁻⁷**cm/s；地下水位>**2m**；黏土>**6m**；距供水井>**300m**；距水源地>**500m** |

### 七、综合评价公式

- 土地损失：L = ΣSᵢ·Pᵢ
- 农作物损失：L<sub>a</sub> = ΣSᵢ(Δyᵢ·Cᵢ+ΔCᵢ·yᵢ)
- 断水损失：L<sub>w</sub> = P₀·Q·L<sub>w</sub>'
- 农业人口价值：B<sub>a</sub> = T<sub>n</sub>ΣSᵢyᵢCᵢ + ΣBⱼTⱼ (T<sub>n</sub>=**45年**)
- 非农人口价值：B<sub>a</sub>' = T<sub>n</sub>'ΣB<sub>t</sub>' (T<sub>n</sub>'=**35年**)

## 第四节 生态环境规划

### 一、生态规划概述

**概念**：以生态学原理为指导，应用系统科学、环境科学等多学科手段辨识、模拟和设计生态系统内部各种生态关系，确定资源开发利用和保护的生态适宜性，探讨改善系统结构和功能的生态对策，促进人与环境系统协调、持续发展的规划方法。

**主要任务3条**：①充分了解自然环境/资源性能及与人类活动关系 ②立足社会经济条件和资源潜力，形成功能互补协调 ③追求总体关系协调，改善自我调控和发展能力

**主要类型**：

| 分类依据 | 类型 |
|----------|------|
| 按地理空间尺度 | ①区域生态规划(编制目录+提出土地利用方案) ②景观生态规划(景观格局与生态过程，强调空间格局控制) ③生物圈保护区规划(保证生物多样性完整性) |
| 按地理和生存环境 | 陆地/海洋/城市/农村生态系统规划 |
| 按社会科学门类 | 经济/人类/民族文化生态规划 |

**步骤5步**：①确定范围目标 → ②收集资料分析适宜性 → ③建立评价分级准则 → ④评价生态适宜性 → ⑤确定方案

### 二、生态规划内容与方法

#### 1. 生态调查

| 内容 | 项目 |
|------|------|
| 自然环境与自然资源 | 气候气象/地理特征/水资源/土壤/野生动植物/珍稀物种/开发历史/自然灾害/演变特征/基础图件 |
| 社会经济 | 社会结构(人口/资源/年龄/生活水平/科技文化) + 经济结构(产业构成/资源利用) |
| 生态环境质量 | 土地利用/植被覆盖/水资源利用/生物多样性 |

**调查方法4种**：①实地调查(最直接，小区域重要) ②历史资料(探索人-环境关系) ③社会调查(公众愿望+专家经验) ④遥感技术(大面积调查有效)

#### 2. 生态分析与评价（5项）

| 项目 | 要点 |
|------|------|
| ①生态过程分析 | 能流/物质循环/水循环/土地承载力/景观格局等与区域发展相关的生态过程 |
| ②生态潜力分析 | 狭义=第一性生产力(初级生产力)，光温水土组合最大生产力=农林牧上限；广义=所有生态资源自然条件下的生产供应能力 |
| ③生态格局分析 | 自然群落→农林群落(基质)；城镇农村(镶嵌体)；交通林带河道(廊道)；残存自然斑块(维护生态/物种/多样性) |
| ④生态敏感性分析 | 评价因子对人类活动反应；包括水土流失/自然灾害风险/特殊价值生态系统/重要集水区评价 |
| ⑤土地质量与区位评价 | 复合生态系统综合归纳；加权综合/主成分分析等方法→土地质量与区位评价图 |

#### 3. 决策分析（3项）

| 内容 | 要点 |
|------|------|
| ①生态适宜性分析 | **生态规划的核心**；根据资源环境性能按发展需求划分适宜性等级；麦克哈格提出图形空间叠置法，GIS技术进一步发展 |
| ②生态功能区划与土地利用布局 | 根据复合生态系统结构功能划分功能单元；综合考虑敏感性/服务功能/城市功能差异→合理分区布局 |
| ③规划方案制定/评价/选择 | 在分析评价基础上制定方案；由子规划构成；目标=促进社会经济+改善生态+增强持续发展能力 |

### 三、城市生态规划

#### 1. 城市生态系统特征（5点）

| # | 特征 | 说明 |
|---|------|------|
| 1 | 人工主导性 | 一切设施人工制造；生产者少；消费者是人；分解者受抑制 |
| 2 | 高度集中性和开放性 | 人口密集；食物/物质/能量需外界输入；废弃物需人工处理 |
| 3 | 不完整性 | 食物能量不能内解；缺分解者→废弃物需人工处理→高成本低效 |
| 4 | 高度依赖性 | 自我更新和初级生产能力低；城市越大依赖越强 |
| 5 | 脆弱性 | 自我调节能力低；切断外部供应即瘫痪；灾害抵抗恢复能力有限 |

#### 2. 城市生态规划与生态城市

- **城市生态规划**：以生态学理论为指导，对城市社会/经济/技术/生态环境全面综合规划，促进良性循环，创造舒适优美清洁安全环境
- **生态城市**：广义=按生态学原则建立的社会经济自然协调新型社会关系；狭义=按生态学原理设计的高效和谐健康可持续人类聚居环境

#### 3. 基本程序（5步）

①明确范围目标 → ②生态要素调查评价预测(按1km×1km或5km×5km划分单元建数据库) → ③生态适宜性分析评价(**核心**) → ④制定方案措施 → ⑤方案评价

#### 4. 主要内容（5项）

| # | 内容 | 要点 |
|---|------|------|

### 四、生态工业园区规划

#### 1. 概念与类型

**概念**：依据清洁生产要求、循环经济理念和工业生态学原理设计建立的新型工业园区。建立"生产者-消费者-分解者"循环途径，物质闭路循环+能量多级利用+废物最小化。

| 类型 | 定义 |
|------|------|
| 行业类园区 | 以某一类工业行业核心企业，通过物质能量集成建立共生关系 |
| 综合类园区 | 不同行业企业组成，在高新技术/经开区基础上改造 |
| 静脉产业类园区 | 以资源再生利用产业企业为主体 |

#### 2. 规划原则

**3R原则**：减量化(Reduce) + 再利用(Reuse) + 资源化(Recycle) + 适用性 + 可操作性

#### 3. 编制程序（6步）

①确定任务(委托有资质单位) → ②调查收集资料(兼顾周边) → ③编制规划大纲(按HJ/T409-2007) → ④编制规划和技术报告 → ⑤规划论证及公众参与(专家论证会) → ⑥规划评审(审查→修改→报批稿)

#### 4. 编制内容（7项）

1. 概况和现状分析
2. 建设必要性分析
3. 建设总体设计
4. 园区主要行业生态工业发展规划
5. 资源循环利用和污染控制规划
6. 重大项目投资与效益分析
7. 建设保障措施

## 自测题与答案

**Q1：大气环境规划的主要内容和类型？**
> 内容6步：查清问题→确定目标→建立模型→制定方案→优选→实施。类型2类：大气环境质量规划（规定功能区浓度限值，管理基础）和大气污染控制规划（实现质量方案，新建城市合理布局/已污染城市寻求经济可行方案）

**Q2：大气污染物总量控制三种方法？**
> ①A-P值法：A值法算区域总允许排放量+P值法分配到每个源 ②平权分配法：基于多源模式按贡献率削减 ③优化方法：环境效益+经济费用结合，治理费用最小方案

**Q3：水功能区划一级区和二级区各有哪些？**
> 一级4类：保护区、缓冲区、开发利用区、保留区。二级7类（在开发利用区内）：饮用水源区、工业用水区、农业用水区、渔业用水区、景观娱乐用水区、过渡区、排污控制区

**Q4：水环境容量与最大允许纳污量的区别？**
> 水环境容量=完全分散排放方式对应的污染物容纳量（极限值）；最大允许纳污量=在排污口空间分布及排放方式给定条件下允许排放的量；容量是允许纳污量的极限值

**Q5：水污染控制最优化法三种类型？**
> ①排污口最优化处理(水质规划)：求解各排放口处理效率最佳组合，费用最低 ②最优化均匀处理(厂群规划)：同一效率下寻求最佳处理厂位置与规模组合 ③区域最优化处理：前二者综合，既考虑最佳位置数量又考虑最佳效率

**Q6：固体废物三大分类及危险废物特性？**
> 三类：城市生活垃圾、工业固废、危险废物。危险废物6特性：急性毒性、易燃性、反应性、腐蚀性、浸出毒性、疾病传染性

**Q7：固体废物处置选址5个条件及关键数字？**
> ①距离：下风向、人畜居栖500m外 ②交通：距公路/铁路/河流≤500m ③环保：使用15-20年、距地表水>100m、距机场>10km ④建设：坡度≤5% ⑤地质：K≤10⁻⁷cm/s、地下水位>2m、黏土>6m、距供水井>300m、距水源地>500m

**Q8：城市生态系统五大特征？**
> ①人工主导性(生产者少/消费者是人/分解者受抑) ②高度集中性和开放性(需外界输入) ③不完整性(食物/能量不能内解/缺分解者) ④高度依赖性(自我更新能力低/城市越大依赖越强) ⑤脆弱性(自我调节能力低/切断外部即瘫痪)

**Q9：生态工业园区的三种类型和3R原则？**
> 类型：行业类(同行业企业共生)、综合类(不同行业/开发区改造)、静脉产业类(资源再生利用)。3R：减量化(Reduce)、再利用(Reuse)、资源化(Recycle)

**Q10：生态适宜性分析的地位和方法？**
> 生态适宜性分析是生态规划的核心。方法：麦克哈格提出图形空间叠置法，随GIS技术发展进一步完善。目标：根据区域自然资源与环境性能，按发展需求划分适宜性等级
