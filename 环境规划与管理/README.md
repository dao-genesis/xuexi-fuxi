## 五 · 识别歧义之记（首跑发现）

> *正言若反。* 章节识别基于 PDF 名信号——以下二处需注：

### 疑① · 第0章（绪论）与 第2章 内容相同

源因：`001/002-part02-绪论-环境规划与管理.pdf` 之 **"part02"** 中的 "02"，被章号识别器视为章号 2。
故 001/002 这两份"绪论 part02"同时被归入 **第0章（绪论）** 与 **第2章** —— `_章节图谱.md` 中两章主版皆显 129 页（同 002）。

**实际**：王老师"绪论"教材分 part01/part02 两部分，"02" 为 part 编号而非章号。

**改进路径**（任择其一）：

- **守愚**：保留现状，复习时自知"第0章≡第2章"，重点关注第2章下；
- **手修**：编辑 `02_解析成果/001-part02-.../_doc.json` 与 `002-part02-.../_doc.json`，将 `chapter_num: 2` 改为 `null` 或 `0`，再跑 `python 课程_闭环.py 环境规划与管理 --rescan`。

### 疑② · 第3章 "环境规划与管理的政策方针法律法规" 未归类

源因：`003-第二三章...-环境规划与管理.pdf` 标题含 "**第二三章**" 之歧义文字，章号识别器无所适从而置 `chapter_num: null`，故归入未归类区。

**实际**：此 PDF 内容跨第2章（环境规划与管理的政策方针）与第3章（环境规划与管理的法律法规）。

**改进路径**：手修 `02_解析成果/003-第二三章.../_doc.json` 中：

```json
{
  "chapter_num": 3,
  "all_chapter_nums": [2, 3]
}
```

再跑 `python 课程_闭环.py 环境规划与管理 --rescan`，第3章节即生。

> *知止所以不殆。* 二疑皆不阻闭环，可后续从容修。
