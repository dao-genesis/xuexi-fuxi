## 上编综述：风云变幻的八十年 (1840-1919)

### 核心概念

- **近代中国社会性质**：半殖民地半封建社会
- **两对主要矛盾**：帝国主义与中华民族的矛盾（最主要）；封建主义与人民大众的矛盾
- **两大历史任务**：争取民族独立、人民解放；实现国家富强、人民富裕
- **关系**：前一任务为后一任务扫清障碍、创造前提

## 第一章 进入近代后中华民族的磨难与抗争

### 第一节 鸦片战争前的中国和世界

- 中国封建社会特点：自给自足的自然经济占主导；专制主义中央集权；儒家思想为正统
- 世界：资本主义快速发展 → 殖民扩张 → 需要市场和原料

### 第二节 西方列强对中国的侵略

**四种侵略方式**：

1. **军事侵略**：发动战争 → 签订不平等条约
   - 1840-1842 第一次鸦片战争 →《南京条约》（割香港岛、五口通商、协定关税、赔款2100万银元）
   - 1856-1860 第二次鸦片战争 →《天津条约》《北京条约》（割九龙半岛、开放更多口岸）
   - 1894-1895 甲午战争 →《马关条约》（割台湾及澎湖、赔款2亿两、允许在华设厂）
   - 1900-1901 八国联军 →《辛丑条约》（赔款4.5亿两、设使馆区、严禁反帝）

2. **政治控制**：领事裁判权（治外法权）；把持海关；勾结清政府镇压人民

3. **经济掠夺**：勒索赔款；控制通商口岸；剥夺关税自主权；商品倾销与资本输出

4. **文化渗透**：披着宗教外衣进行侵略活动；为侵略制造舆论

### 第三节 反抗外来侵略的斗争

- 人民群众：三元里人民抗英；义和团运动（粉碎瓜分图谋）
- 爱国官兵：关天培、陈化成、邓世昌、冯子材（镇南关大捷）

### 第四节 反侵略斗争的失败与民族意识的觉醒

**失败原因**（★必考）：
1. **社会制度腐败**（根本原因）
2. **经济技术落后**（重要原因）

**民族意识觉醒**：
- "师夷长技以制夷"（魏源《海国图志》）
- "救亡"口号（严复《天演论》"物竞天择，适者生存"）
- 孙中山"振兴中华"

## 第二章 不同社会力量对国家出路的早期探索

### 第一节 太平天国农民战争

- 1851金田起义 → 1853定都天京 → 1864天京陷落
- 《天朝田亩制度》：绝对平均主义 → 无法实现
- 《资政新篇》：中国近代第一个比较系统的发展资本主义方案 → 未实施
- **意义**：沉重打击清王朝和外国侵略势力
- **失败原因**：农民阶级局限性（无先进阶级领导、无科学纲领）；战略失误；中外反动势力联合镇压
- **教训**：农民阶级不能充当中国革命的主力军和领导力量

### 第二节 洋务运动

- 时间：19世纪60-90年代
- 口号：前期"自强"（军事工业）；后期"求富"（民用工业）
- 代表：奕䜣、曾国藩、李鸿章、左宗棠、张之洞
- 内容：兴办近代企业；建立新式海陆军；创办新式学堂、派遣留学生
- **性质**：地主阶级自救运动
- **失败标志**：甲午战争北洋水师覆没
- **失败原因**：封建性（官办/官督商办）；对外依赖性；管理腐朽
- **历史作用**：客观上促进了中国早期工业和民族资本主义发展；是中国近代教育的开端

### 第三节 戊戌维新运动

- 1898年（戊戌年），103天 → "百日维新"
- 代表：康有为、梁启超、谭嗣同
- 内容：政治上兴民权、设议院、定宪法；经济上发展资本主义；文教上改革科举
- **性质**：资产阶级性质的改良运动
- **失败原因**：民族资产阶级软弱性 → 不敢否定封建主义 → 不敢发动群众 → 对帝国主义抱幻想
- **意义**：资产阶级登上政治舞台的第一次表演；起了思想启蒙作用
- **教训**：自上而下的改良道路在中国行不通

## 第三章 辛亥革命与君主专制制度的终结

### 第一节 举起近代民族民主革命的旗帜

- 资产阶级革命派形成：孙中山、黄兴、章炳麟
- **三民主义**（★必考）：
  - 民族主义：驱除鞑虏，恢复中华（推翻清朝统治，争取民族独立）
  - 民权主义：创立民国（推翻封建君主专制，建立资产阶级民主共和国）→ **核心**
  - 民生主义：平均地权（资本主义土地纲领）
- 革命与改良的辩论：要不要以革命手段推翻清王朝；要不要推翻帝制实行共和；要不要社会改革

### 第二节 辛亥革命与建立民国

- 1911.10.10 武昌起义
- 1912.1.1 中华民国临时政府成立 → 孙中山临时大总统
- 1912.2.12 清帝退位 → **两千多年封建君主专制制度终结**
- 《中华民国临时约法》：中国历史上第一部具有资产阶级共和国宪法性质的法典

### 第三节 辛亥革命的失败

- 袁世凯篡夺革命果实 → 北洋军阀统治
- **辛亥革命意义**（★必考）：
  1. 推翻了封建君主专制制度 → 伟大历史性巨变
  2. 使民主共和观念深入人心
  3. 促使社会经济、思想习惯和社会风俗的变化
  4. 推动亚洲民族解放运动
- **失败原因**（★必考）：
  1. **根本原因**：在帝国主义时代，半殖民地半封建的中国，资本主义建国方案行不通
  2. **主观原因**：民族资产阶级软弱性和妥协性 → 不敢彻底反帝反封建 → 不敢发动群众
- **教训**：资产阶级共和国方案在中国行不通；旧民主主义革命走到尽头

## 中编综述：翻天覆地的三十年 (1919-1949)

- 三种政治力量、三种建国方案：
  1. 地主阶级和买办性大资产阶级 → 军事独裁专政
  2. 民族资产阶级 → 资产阶级共和国
  3. 工人阶级、农民阶级和城市小资产阶级 → 人民民主专政
- 两个中国之命运：光明与黑暗

## 第四章 开天辟地的大事变

### 第一节 新文化运动和五四运动

**新文化运动**：
- 1915 陈独秀《青年杂志》（后改名《新青年》）→ 标志新文化运动开始
- 口号：民主（德先生）和科学（赛先生）
- 内容：提倡民主、科学，反对专制、迷信；提倡新道德，反对旧道德；提倡新文学，反对旧文学
- 意义：空前思想解放运动；为马克思主义传播创造条件

**五四运动**（★必考）：
- 导火线：巴黎和会上中国外交失败
- 1919.5.4 北京学生 → 6月5日上海工人罢工 → 工人阶级登上政治舞台
- **历史意义**：中国新民主主义革命的开端
  1. 表现了反帝反封建的彻底性
  2. 是一次真正的群众运动
  3. 促进了马克思主义在中国的传播及其与中国工人运动的结合
  4. 发生在俄国十月革命之后 → 属于世界无产阶级社会主义革命的一部分

### 第二节 马克思主义进一步传播与中国共产党诞生

- 早期马克思主义者：李大钊（中国宣传马克思主义第一人）、陈独秀
- 三次思想论争：问题与主义论争；关于社会主义的论争；与无政府主义的论争

**中国共产党成立**（★必考·"开天辟地的大事变"）：
- 1921.7.23 中共一大（上海 → 嘉兴南湖）
- 纲领：推翻资产阶级政权；承认无产阶级专政；消灭资本家私有制
- **成立意义**：
  1. 新的坚强的领导核心
  2. 新的革命方法（群众路线）
  3. 新的奋斗目标（社会主义和共产主义）
  4. 新的革命前途（社会主义方向）
  5. 使中国革命有了可信赖的组织者和领导者

- 1922 中共二大：**第一次提出反帝反封建的民主革命纲领**
  - 最高纲领：实现社会主义、共产主义
  - 最低纲领：打倒军阀、推翻国际帝国主义压迫、统一中国为真正民主共和国

### 第三节 中国革命的新局面

- 1924-1927 第一次国共合作（中共三大决定以个人名义加入国民党）
- 北伐战争胜利
- **大革命失败**：1927 四一二政变 → 七一五政变
- **失败原因**：中共领导机关犯了右倾机会主义错误 → 放弃无产阶级领导权和武装力量
- **经验教训**：必须坚持无产阶级领导权；必须结成巩固的工农联盟；必须建立革命武装

## 第五章 中国革命的新道路

### 第一节 对革命新道路的艰苦探索

- 1927 国民党在全国统治的建立（宁汉合流 → 南京政府）
- **南昌起义**（1927.8.1）：打响武装反抗国民党反动派第一枪 → 人民军队诞生
- **八七会议**（1927.8.7）："枪杆子里面出政权"；土地革命和武装反抗总方针
- 秋收起义 → 三湾改编（党指挥枪原则）→ 井冈山革命根据地
- **农村包围城市、武装夺取政权**道路理论形成
  - 《中国的红色政权为什么能够存在？》
  - 《井冈山的斗争》
  - 《星星之火，可以燎原》

### 第二节 中国革命在探索中曲折前进

- 土地革命：打土豪、分田地 → 《井冈山土地法》→《兴国土地法》
- **遵义会议**（1935.1）（★必考）：
  - 纠正了王明"左"倾军事错误
  - 确立毛泽东在党中央的实际领导地位
  - **中国共产党历史上一个生死攸关的转折点**
  - 标志中国共产党在政治上走向成熟
- 长征（1934.10-1936.10）：红军三大主力会师 → 长征胜利
- 抗日民族统一战线策略的制定：《八一宣言》→ 瓦窑堡会议

## 第六章 中华民族的抗日战争

### 第一节 日本发动灭亡中国的侵略战争

- 1931 九一八事变 → 侵占东北
- 1937 七七事变（卢沟桥事变）→ 全面侵华
- 南京大屠杀：30万以上中国平民和战俘被害

### 第二节 从局部抗战到全国性抗战

- 一二九运动（1935.12.9）：促进民族觉醒
- 西安事变（1936.12.12）：张学良、杨虎城 → 和平解决 → 时局转换枢纽
- 1937.9 国民党公布中共提出的合作宣言 → 抗日民族统一战线正式形成

### 第三节 国民党正面战场

- 淞沪会战、忻口会战、徐州会战（台儿庄大捷）、武汉会战
- 失败原因：片面抗战路线（不发动群众）；战略指导失误；敌强我弱

### 第四节 共产党敌后战场 → 中流砥柱

- 平型关大捷（1937.9）：抗战以来第一次大捷
- 百团大战（1940）：敌后战场最大规模战役
- **共产党是抗战中流砥柱**（★必考）：
  1. 倡导和坚持抗日民族统一战线
  2. 坚持全面抗战路线
  3. 开辟敌后战场
  4. 推进马克思主义中国化 → 毛泽东思想成熟

### 第五节 抗日战争的胜利

- 1945.8.15 日本宣布无条件投降 → 9.2 签署投降书
- **胜利原因**（★必考）：
  1. 中国共产党在全民族抗战中发挥了中流砥柱作用
  2. 以国共合作为基础的抗日民族统一战线
  3. 全民族抗战
  4. 国际反法西斯力量的支持
- **意义**：近代以来中华民族反抗外敌入侵第一次取得完全胜利；中华民族由衰败走向复兴的重大转折

  1. 坚持马克思列宁主义普遍真理与中国具体实际相结合 → 毛泽东思想

  3. 建立广泛的统一战线
  4. 坚持共产党领导 → 根本保证

## 第八章 中华人民共和国成立与社会主义建设道路的探索

### 第一节 从新民主主义向社会主义过渡的开始

- 新中国成立初期三大任务：继续完成民主革命遗留任务；恢复国民经济；巩固新生政权
- 土地改革（1950-1952）：废除封建土地制度 → 农民成为土地主人
- 抗美援朝（1950-1953）：保家卫国 → 提高中国国际地位

### 第二节 社会主义道路：历史和人民的选择

- 过渡时期总路线（1953）："一化三改"（★必考）
  - 一化：社会主义工业化（主体）
  - 三改：对农业、手工业、资本主义工商业的社会主义改造（两翼）
  - 特点：建设与改造同时并举

### 第三节 有中国特点的向社会主义过渡的道路

- 农业改造：互助组 → 初级社 → 高级社
- 资本主义工商业改造：**和平赎买**（世界社会主义运动史上的创举）
- 社会主义改造完成（1956）→ **社会主义制度在中国确立**（★必考·意义）：
  1. 中国历史上最深刻最伟大的社会变革
  2. 为当代中国一切发展进步奠定了根本政治前提和制度基础
  3. 创造性完成了由新民主主义到社会主义的转变

### 第四节 社会主义建设在探索中曲折发展

- **中共八大**（1956）（★必考）：
  - 正确分析国内主要矛盾：人民对于经济文化迅速发展的需要与当前经济文化不能满足人民需要的状况之间的矛盾
  - 确定主要任务：集中力量发展社会生产力
  - 坚持既反保守又反冒进的经济建设方针

- 探索中的失误：大跃进、人民公社化运动 → "左"倾错误发展
- **文化大革命**（1966-1976）：严重曲折 → 教训深刻

- **建设成就**（不能因错误而否定）：
  - 独立完整的工业体系和国民经济体系
  - "两弹一星"
  - 外交突破（恢复联合国合法席位、中美建交）

## 第九章 改革开放与中国特色社会主义

### 第一节 历史性的伟大转折

- **十一届三中全会**（1978.12）（★必考·"伟大历史转折"）：
  1. 彻底否定"两个凡是"的错误方针 → 重新确立实事求是的思想路线
  2. 停止"以阶级斗争为纲" → 工作重心转移到经济建设上来
  3. 作出改革开放的伟大决策
  4. 形成以邓小平为核心的党中央领导集体
  - **意义**：中华人民共和国成立以来具有深远意义的伟大转折；开启了改革开放和社会主义现代化建设新时期

### 第二节 改革开放的展开

- 农村改革先突破：家庭联产承包责任制 → 乡镇企业异军突起
- 经济特区：深圳、珠海、汕头、厦门（1980）→ 海南（1988）
- 中共十二大（1982）："建设有中国特色的社会主义"
- 中共十三大（1987）：社会主义初级阶段理论；"一个中心、两个基本点"基本路线

### 第三节 改革开放的新阶段

- 邓小平南方谈话（1992）：社会主义本质；三个有利于标准
- 中共十四大（1992）：建立社会主义市场经济体制

## 期末高频简答/论述题精选

### 1. 为什么说辛亥革命是近代中国的一次伟大历史性巨变？又为什么说它失败了？

**巨变**：推翻了封建君主专制制度（非封建制度整体）；使民主共和观念深入人心；促进了社会经济思想风俗变化；推动了亚洲民族解放运动。

**失败**：没有完成反帝反封建的任务；革命果实被袁世凯篡夺。根本原因：在帝国主义时代，半殖民地半封建的中国，资本主义建国方案行不通。主观原因：民族资产阶级软弱性妥协性。

### 2. 为什么说中国共产党的成立是开天辟地的大事变？

1. 新的坚强的领导核心
2. 新的革命方法（群众路线）
3. 新的奋斗目标（社会主义和共产主义）
4. 新的革命前途（社会主义方向）
5. 使中国革命面貌焕然一新

### 3. 为什么说中国共产党是抗日战争的中流砥柱？

1. 倡导、促成并坚持抗日民族统一战线
2. 坚持全面抗战路线（人民战争）
3. 开辟和坚持敌后战场
4. 推进马克思主义中国化 → 毛泽东思想走向成熟

### 4. 为什么说"没有共产党就没有新中国"？中国革命取得胜利的基本经验是什么？

**原因**：共产党领导中国人民走新民主主义道路，推翻三座大山，建立了新中国。

**基本经验（三大法宝）**：
1. 统一战线
2. 武装斗争
3. 党的建设

### 5. 为什么说十一届三中全会是中华人民共和国成立后的伟大历史转折？

1. 重新确立实事求是的思想路线 → 否定"两个凡是"
2. 停止"以阶级斗争为纲" → 工作重心转移到经济建设
3. 作出改革开放伟大决策
4. 形成以邓小平为核心的第二代领导集体
5. 结束了党和国家工作在徘徊中前进的局面

### 6. 简述近代中国社会的基本特征。

1. 资本-帝国主义逐步控制中国政治经济
2. 封建势力成为帝国主义统治中国的社会支柱
3. 自然经济遭破坏但封建剥削制度依然存在
4. 中国各地经济政治文化发展极不平衡
5. 中国人民贫困和不自由程度世界罕见

### 7. 简述反侵略斗争失败的原因。

1. **社会制度腐败**（根本原因）：清政府腐败无能 → 不敢发动群众
2. **经济技术落后**（重要原因）：武器装备落后、军队素质低下

### 8. 简述遵义会议的历史意义。

1. 纠正了王明"左"倾军事错误
2. 确立毛泽东在党中央的实际领导地位
3. 中国共产党历史上生死攸关的转折点
4. 标志中国共产党在政治上走向成熟（独立自主解决自身问题）

### 9. 简述过渡时期总路线的内容及特点。

**内容**："一化三改"——社会主义工业化（主体）；对农业、手工业、资本主义工商业的社会主义改造（两翼）。

**特点**：社会主义建设与社会主义改造同时并举。

### 10. 简述中共八大的主要贡献。

1. 正确分析国内主要矛盾
2. 确定主要任务：集中力量发展社会生产力
3. 坚持既反保守又反冒进的经济建设方针
4. 提出扩大民主、健全法制任务

## 辨析题答题要领

辨析题 = 先判断对错 + 再分析原因

**常见辨析角度**：
1. "辛亥革命推翻了封建制度" → ✗ 推翻的是封建**君主专制制度**，非整个封建制度
2. "洋务运动是资产阶级性质的改革" → ✗ 地主阶级自救运动
3. "五四运动是旧民主主义革命" → ✗ 新民主主义革命的开端
4. "中国革命的中心问题是农民问题" → ✓ 农民是中国革命的主力军
5. "社会主义改造就是取消资本主义" → ✗ 是和平赎买、逐步改造
