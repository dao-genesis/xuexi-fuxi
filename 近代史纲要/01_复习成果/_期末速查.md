## 数字速查

- **两个矛盾**：帝国主义↔中华民族；封建主义↔人民大众
- **两个任务**：民族独立人民解放；国家富强人民富裕
- **三民主义**：民族、民权（核心）、民生
- **三大法宝**：统一战线、武装斗争、党的建设
- **三大作风**：理论联系实际、密切联系群众、批评与自我批评
- **三条道路**：独裁专政、资产阶级共和国、人民民主专政
- **四个选择**：马克思主义、中国共产党、社会主义、改革开放
- **五位一体**：经济、政治、文化、社会、生态
- **四个全面**：小康社会、深化改革、依法治国、从严治党
- **五大发展理念**：创新、协调、绿色、开放、共享
- **中国式现代化五个特征**：人口规模巨大、共同富裕、两个文明协调、人与自然和谐、和平发展

## 原因速查

| 问题 | 原因 |
|------|------|
| 反侵略失败 | 根本：社会制度腐败；重要：经济技术落后 |
| 太平天国失败 | 农民阶级局限性、无先进阶级领导 |
| 洋务运动失败 | 封建性、依赖性、腐朽性 |
| 戊戌维新失败 | 民族资产阶级软弱性→不敢反帝反封建 |
| 辛亥革命失败 | 根本：资本主义方案行不通；主观：资产阶级软弱性 |
| 大革命失败 | 右倾机会主义→放弃领导权和武装 |
