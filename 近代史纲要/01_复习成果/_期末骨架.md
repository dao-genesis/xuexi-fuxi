# 中国近现代史纲要 · 期末骨架

> 骨架者，道之骨也。一气贯通，万变不离其宗。

---

## 一条主线

**四个选择**：马克思主义 → 中国共产党 → 社会主义道路 → 改革开放

## 两大历史任务

- 争取民族独立、人民解放（前提）
- 实现国家富强、人民富裕（目的）

## 两对主要矛盾

- 帝国主义与中华民族（最主要）
- 封建主义与人民大众

## 三个时期

| 编 | 时期 | 时间 | 主题 |
|---:|------|------|------|
| 上编 | 旧民主主义革命 | 1840-1919 | 探索→失败→旧路不通 |
| 中编 | 新民主主义革命 | 1919-1949 | 党领导→胜利→新中国 |
| 下编 | 社会主义建设 | 1949- | 改造→探索→改革→新时代 |

## 十章骨架

```
第1章 鸦片战争 → 半殖民地半封建 → 反侵略失败(制度腐败+技术落后)
第2章 太平天国/洋务/维新 → 农民/地主/资产阶级探索 → 全失败
第3章 辛亥革命 → 推翻君主专制 → 但资本主义方案行不通
第4章 五四运动 → 新民主主义开端 → 党成立(开天辟地)
第5章 南昌起义 → 农村包围城市 → 遵义会议(生死转折)
第6章 抗日战争 → 统一战线 → 共产党中流砥柱 → 胜利
第7章 解放战争 → 三大战役 → 新中国成立(没有共产党就没有新中国)
第8章 社会主义改造 → 一化三改 → 制度确立(最深刻伟大变革)
第9章 十一届三中全会 → 改革开放 → 中国特色社会主义
第10章 十八大 → 新时代 → 中国梦 → 中国式现代化
```

## 六大转折点

1. **1840** 鸦片战争 → 近代史起点
2. **1919** 五四运动 → 新民主主义革命开端
3. **1921** 中共成立 → 开天辟地
4. **1935** 遵义会议 → 生死攸关转折
5. **1949** 新中国成立 → 民族独立人民解放
6. **1978** 十一届三中全会 → 伟大历史转折

## 三大法宝

统一战线 + 武装斗争 + 党的建设

## 五次探索失败教训

| 探索 | 阶级 | 失败原因 |
|------|------|----------|
| 太平天国 | 农民 | 无先进阶级领导、无科学纲领 |
| 洋务运动 | 地主 | 封建性、依赖性、腐朽性 |
| 戊戌维新 | 资产阶级(改良) | 软弱性→不敢反帝反封建→不敢发动群众 |
| 辛亥革命 | 资产阶级(革命) | 软弱性→资本主义方案行不通 |
| 大革命 | 国共合作 | 右倾机会主义→放弃领导权和武装 |

## 新疆考点骨架

- 1949.9 新疆和平解放（陶峙岳、包尔汗起义）
- 1954.10 新疆生产建设兵团成立
- 1955.10.1 新疆维吾尔自治区成立
- 中国共产党在新疆的领导地位 = 历史必然 + 人民选择
