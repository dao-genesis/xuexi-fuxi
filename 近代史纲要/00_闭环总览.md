## 六 · 期末高频考点

1. **近代中国社会性质**：半殖民地半封建 → 两对主要矛盾 → 两大历史任务
2. **反侵略斗争失败原因**：社会制度腐败 + 经济技术落后
3. **太平天国/洋务/维新/辛亥**：各阶级探索 → 均失败 → 旧路不通
4. **辛亥革命意义与失败**：伟大巨变但未完成反帝反封建 → 资产阶级共和国方案行不通
5. **五四运动**：新民主主义革命开端 → 马克思主义传播 → 党的成立
6. **中国共产党成立**：开天辟地的大事变 → 革命面貌焕然一新
7. **农村包围城市道路**：马克思主义中国化 → 遵义会议 → 长征
8. **抗日战争**：国民党正面战场 + 共产党敌后战场 → 中流砥柱
9. **解放战争**：三条道路/两种命运 → 没有共产党就没有新中国
10. **社会主义改造**：一化三改 → 创造性转变 → 社会主义制度确立
11. **改革开放**：十一届三中全会 → 伟大转折 → 中国特色社会主义
12. **新时代**：十八大 → 中国梦 → 五位一体/四个全面 → 中国式现代化
